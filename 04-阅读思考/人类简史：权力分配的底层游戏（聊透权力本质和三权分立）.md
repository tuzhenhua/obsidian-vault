
# 权力的底层逻辑：铁三角 + 三权分立

> 妈妈常说：**权和钱是命相连。** 六个字，比任何理论都直接。权没了钱就散了，钱没了权就塌了——不是因果关系，是共生关系。
>
> 原文为公众号三篇连载，合并精简，只保留核心框架。

---

## 一、权力的铁三角

一个完备的权力由三根支柱构成：

```
权力的根基是破坏力  —— 没有惩罚威慑，一切权力都是废纸
权力的命脉是信息权  —— 看不见就管不了
权力的抓手是控制力  —— 能按自己意志行动
```

### 信息权——权力的眼睛

**案例：居委会投票。** 到了现场才发现不认识任何候选人，事先要名单被拒绝。投票环节走进"小房间"，桌上放着几个名字供"抄写"。投完发现：没有信息权的投票权，是一张被阉割的废纸。

同样的逻辑：**越级汇报 = 掠夺中层领导的信息权**。下属绕过你向上汇报，你的眼睛被蒙住——对下看不清，对上说不明。古代"皇权不下县"，也是因为县级信息根本上不去。

### 控制力——权力的手

你能不能让事情按你的意志发生。领导说话不好使，通常是因为没有奖惩权（控制力+破坏力），发号施令像放空枪。

### 破坏力——权力的根

没有国家机器提供暴力威慑，法律规定的投票权就是废纸。破坏力是权力最底层的担保——你可以不用，但不能没有。

### 极致的权力 ≠ 极致的自由

如果三项能力都到极致：天下之事尽知（信息权）、天下之人尽控（控制力）、天下之物尽有（破坏力）——这看起来是极致自由。但古代皇帝最接近这个位置，反而陷入了极致的不自由：吃饭每样只能一口，作息被严格管制。因为权力太大，官僚和百姓没有安全感，必须反过来牢牢限制住皇权。

## 二、权力的暗线：权责平衡

> 权力与责任必须动态平衡。彻底失衡，小到家庭、大到国家，都会崩溃。

| 失衡方向 | 案例 | 结果 |
|---------|------|------|
| **责大权小** | 张三被安排巨大业绩指标，但调动不了任何资源 | 离职 |
| **权大责小** | 小出纳权限亿级，监管缺位，轻松转走两千万跑路 | 资金流失 |
| **权大责小（系统级）** | 皇帝可随意兴兵、杀人、挥霍，最多下个罪己诏 | 朝代崩溃 |

权责平衡是权力能否**持久**的关键。铁三角让权力"能用"，权责平衡让权力"能久"。

## 三、权力分配的底层逻辑：三权分立

透过所有权力分配现象，可以抽象出三层：

| 层级 | 权力类型 | 政府映射 | 企业映射 |
|------|---------|---------|---------|
| **第一层** | 规则制定权 | 立法 | 董事会 / 战略部 / 财务部 |
| **第二层** | 执行权 | 行政 | 经营单位 / 事业部 |
| **第三层** | 评判权 | 司法 | 审计 / 风控 |

为什么必须分开？**因为无法杜绝人性中的自私。** 古代皇帝三权集于一身，结果是掌权者富得流油，穷人饿到易子而食。三权分开才能互相制衡——掌权者无法直接用权力获利，就倾向于把蛋糕做大来间接获利。

### 为什么不是"公平"而是"效率"？

权力分配的第一原则不是公平，是**效率**。高考不是绝对公平（随机分配名额），而是机会公平——"让资源自由流动到效率最高的地方"。绝对公平 = 没人努力 = 社会退步。

### 零和博弈 vs 正和博弈

- **坏的权力分配** → 激化零和博弈（只占便宜不分享，每段关系都是一锤子买卖，没有积累）
- **好的权力分配** → 促进正和博弈（合作做大蛋糕，积累声望，增长整体财富）

## 四、普通人也需要"分权"

这套框架可以向下应用到个人：

**家庭财务安全：**
- 规则制定权：大额支出须家庭成员共同决定
- 执行权：不给小孩或不靠谱的人支配大额资金的权限
- 评判权：花错了就收回执行权

**个人自律：**
- 理性制定自己的纪律（规则制定）→ 执行时遵循（执行权）→ 打破就反思惩罚，做好就奖励（评判权）

## 五、快速分析任何权力场景

拿到任何一个组织或关系，问四组问题：

```
1. 信息权：谁掌握了关键信息？谁的眼睛被蒙住了？
2. 控制力+破坏力：谁能调动资源？谁有奖惩手段？谁什么都没有？
3. 权责平衡：权力和责任匹配吗？失衡方向是什么？
4. 三权分立：规则谁定？谁执行？谁评判？三者是分开还是集中？
```

这套框架不判断"对错"，只分析"结构"。结构对了，好人能做好事；结构错了，好人也会被逼成坏人。
