## NAS 实用功能清单

来源：比特荒漠。五个真正高频使用的功能。

### 一、相册同步
- **Immich** — 开源照片服务，AI人脸识别、地理标记、分享链接
- **PhotoSync** — 手机相册自动同步，设一次不用再管
- 核心价值：手机空间彻底解放，家人独立备份空间

### 二、下载服务
- PT / BT / 磁力链 — 睡前丢任务，醒来下好
- **Alist** — 聚合百度网盘、阿里云盘、115、Google Drive、OneDrive 到同一界面，映射为网络驱动器

### 三、AdGuard Home — 全屋广告拦截
- NAS上装 AdGuard Home，路由器DNS指向NAS IP
- 全屋设备（手机、平板、电视、电脑）自动屏蔽广告
- 自部署，不依赖第三方，隐私更安全

### 四、Bitwarden — 自建密码管理器
- 自托管，密码加密存在本地硬盘
- 浏览器插件+手机App自动填密码
- 数据完全自主，不被第三方审查或泄漏

### 五、Docker 实用服务
| 服务 | 用途 |
|------|------|
| **Navidrome** | 私人音乐服务器 |
| **AdGuard Home** | 全屋广告拦截 |
| **Alist** | 网盘聚合 |
| **Home Assistant** | 智能家居中枢，接入米家实现自动化 |
| **Jellyfin / Emby** | 私人媒体库，电视直接播放 |

### 核心总结
> 相册备份 + 下载服务 + 广告拦截 + 密码管理 + 媒体中心
>
> NAS最大的价值不是硬件，是让你对数据的态度从"随便存"变成"好好管"。
