# RustDesk——永久免费的远程桌面方案

> 开源、不限时、全平台。平替 TeamViewer / 向日葵 / ToDesk。只要两台设备能上网，随时随地连回家里或办公室电脑。

---

## 安装

两台电脑都去 [rustdesk.com](https://rustdesk.com) 下载客户端。

| 注意点 | 说明 |
|--------|------|
| 被控端装普通版，不要 Slim 版 | Slim 版缺少服务组件，无法无人值守 |
| 安装时选 "Install for all users" | 否则开机自启可能不生效 |
| 控制端便携版也行 | 笔记本/手机不需要开机自启 |

---

## 被控端配置（四步）

**① 设固定密码**：设置 → 安全 → 密码 → 改为固定密码。不设的话每次连接需要有人在旁边看随机码。

**② 开 OTP 二次验证**：同一页面开启，绑定 authenticator 应用。密码泄露也不怕。

**③ 开机自启**：
- Windows：管理员 PowerShell 执行 `.\rustdesk --install-service`
- Linux：`sudo systemctl enable rustdesk`
- macOS：设置 → 常规 → 勾选开机启动

**④ 记下 ID**：主界面那串数字（如 `123 456 789`），连接时用。

---

## 从外网连入

控制端输入被控端 ID → 点连接 → 输固定密码（开了 OTP 还要输动态码）→ 屏幕出现。就这些步骤。

**手机端**：Android 可双向，iOS 只能控别人。应急够用，长时间办公体验一般。

---

## 网络环境

| 网络类型 | 效果 |
|----------|------|
| 公网 IP 宽带 | ✅ 最佳 |
| 大内网 / CGNAT | ⚠️ 能用但卡，走官方中继 |
| 公司网络 | ⚠️ 可能被防火墙拦截 |
| 5G/4G 热点 | ✅ 轻度操作 |

卡顿解决：画质改"流畅优先"、分辨率降到 1280×720、关音频、降帧率到 10-15fps。

---

## 安全

| 风险 | 应对 |
|------|------|
| 固定密码暴露 | 开 OTP |
| 数据经第三方服务器 | 自建中继（有公网 VPS 的话） |
| 被控端无人值守被连 | 密码设 8 位以上字母数字混排 |

---

## 自建中继（进阶）

有公网 VPS → 放行端口 21115-21117 → 两条命令安装 → 客户端填 VPS IP。延迟从 200ms 降到 20ms 以内。

## 速查

| 场景 | 难度 |
|------|:----:|
| 家里 ↔ 出门笔记本 | ⭐ |
| 手机应急操作电脑 | ⭐ |

| 公司电脑需远程 | ⭐⭐⭐ 先确认防火墙 |
| 远程看视频/做设计 | ❌ 不适合 |
