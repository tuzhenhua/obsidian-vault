# 实弹射击记录

多年前的一次实弹打靶，国外手枪型号。打完后再没玩过实弹射击。

---

## 第一轮

10发，**95.9环**

9 + 9.2 + 9.3 + 9.3 + 9.7 + 9.8 + **9.9×4**

10发中4发接近满环（9.9），最低一发9.0。手稳，准度高。

## 第二轮

10发，**92.9环**

9×4 + 9.2 + 9.3×2 + 9.4 + 9.6 + 9.8

开局4发9.0，后面逐步调回。有调整能力。

---

## 总成绩

**95.9 + 92.9 = 188.8环（20发）**
