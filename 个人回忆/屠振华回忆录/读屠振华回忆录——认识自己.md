# 读屠振华回忆录——认识自己

> 把写出来的东西再读一遍，不只是看故事，是看自己这个人是怎么长成的。
> 童年篇写完了，以后还会有少年、青年。这篇文章就跟着一起长。
> 没事就翻翻，对照一下——有没有老毛病又犯了，有没有进步一点点。
>
> **数据来源**：本文分析的能力和特质，基础数据来自 [[我的真东西清单]]。本文是应用层——把已验证的认知资产展开为诊断工具和行动方案。

## 一、与生俱来的东西

翻完整篇回忆录，有些东西很早就露头了，不是谁教的，是带着的。这些东西是底子的底子，改不了，也不用改，但要认得它们。

### 好奇心

**最早的样子：** 在西牌楼那个7平米的小黑屋里，躺在床上盯着墙壁上的斑驳痕迹发呆。墙上的木房子模型落了灰，脏兮兮的挂在楼梯墙上，但我每次看到都在想——"怎么做得这么精致？什么时候我们能住进这样的房子里？"不是看一眼就过去了，是一直想。后来在科大幼儿园，警察来画小偷的脚印，别的小孩害怕，我蹲在旁边看得入神——"原来破案还有这么一手功夫。"

**这些年长成了什么样：** 什么东西都想知道一点。从BASIC编程到原子弹原理到生物化学，小时候科大家里的那些读物什么都翻；后来订《奥秘》杂志，外星人、金字塔、百慕大三角，每一期都像打开一扇门。到现在还是这样——对不懂的东西第一反应不是躲，是想弄清楚。

**它带来的好处：**
- 学东西不费劲，因为有内在驱动
- 不会无聊，总能找到有意思的事
- 看问题的角度有时候比别人新——因为"灯泡长毛了"这种话，没有好奇心的人说不出来

**它带来的麻烦：**
- 兴趣太散，不容易在一个地方扎下去
- 有时候为了满足好奇心，嘴比脑子快——忍不住要说出来，说出来就惹事

**对照检查清单：**
- [ ] 最近有没有对什么新东西产生兴趣？
- [ ] 这个兴趣是认真了解了一下就过了，还是深入进去研究了？
- [ ] 有没有因为好奇说了不该说的话？
- [ ] 有没有因为好奇做了不该做的事？

### 嘴欠

**最早的样子：** 自然课上老师在黑板上画了个灯泡，周围画一圈光，问"这是什么"。同学们都说通电后灯泡亮了。我嘴欠，偏要说——"灯泡长毛了。"全班笑翻，老师把我赶出去反省。我不是想捣乱，是真的忍不住。这个差别很重要——不是坏，是管不住。

**这些年长成了什么样：** 在科大小孩面前也是这样——满嘴长沙痞话一串串往外冒，人家听着新鲜，但也容易惹毛人。有回不知道说了什么，一个小伙伴追着我从坡上跑到坡下、从这栋楼绕到那栋楼，打又打不过，甩又甩不掉。不是我要惹他，是话说出去了才知道不该说。

**它带来的好处：**
- 不怕事，敢说话。到了新地方不会缩着
- 有幽默感，能把人逗笑——小时候唱跑调的《刘三姐》把大人们笑翻了
- 不憋着，情绪有个出口

**它带来的麻烦：**
- 话出口了才发现不该说，但收不回来了
- 容易得罪人，尤其是得罪那些不了解你的人
- 在正式场合或者重要关系里，这个习惯容易坏事

**对照检查清单：**
- [ ] 今天有没有哪句话说出口就后悔了？
- [ ] 说那句话之前，有没有哪怕一秒钟的停顿？
- [ ] 同样的意思，有没有更稳妥的表达方式？
- [ ] 对面坐着的人，吃得住你这一套吗？

### 敏感

**最早的样子：** 被关在小房子里那个下午——窗外的太阳慢慢挪，屋里的影子慢慢变，门缝里透进来的光，憋尿的滋味——每一丝感觉都记得清清楚楚。妈妈从后门偷看的那一眼，她躲起来的样子，老师拽着我说"你妈妈没来"——那种"妈妈就在门后面但就是够不到"的感觉，几十年了还在。路灯下她让我回去的表情——"她不是不要我，她是觉得那些事不该让孩子看见"——一个几岁的小孩能读出这个，不是谁教的。

**这些年长成了什么样：** 敏感的人疼比别人疼得久。被弹弓的铁丝打了，皮破了，疼完了还记得那种疼。被自行车碾过，吓完了还记得那种吓。但敏感的人也记得住那些好的——白糖汽水玻璃瓶上凝着的水珠、算盘噼里啪啦的声音在安静的晚上特别好听、妈妈坐在对面不说话的样子。那些东西也是敏感的人才能留到现在的。

**它带来的好处：**
- 能察觉到别人注意不到的东西——人的情绪、氛围的变化、细节里的信息
- 有同理心，能理解别人的处境——比如理解妈妈不是不想笑，是"笑是需要余裕的"
- 写作和表达的本钱——没有敏感，写不出这篇回忆录

**它带来的麻烦：**
- 太容易察觉到别人的情绪，有时候会过度解读
- 别人的一句话、一个表情，能想很久
- 累。敏感是很消耗能量的——同样一个场合，别人过去了，你还在消化

**对照检查清单：**
- [ ] 最近有没有因为别人的一句话或一个表情想太多？
- [ ] 那个"想太多"有几分是事实、几分是自己加的戏？
- [ ] 有没有因为太敏感而没做该做的事、没说该说的话？
- [ ] 有没有用敏感做成了什么好事——比如写了点什么、帮了谁？

### 主观意愿强

**最早的样子：** 学校汇演，我领唱《我的中国心》。家里准备了小西装，老师觉得好，让我穿。我死活不穿，就要和同学们穿一样的白衬衫、红领巾。老师劝了半天，拿我没办法。"我的主观意愿太强了，老师都没辙。"——这是自己在回忆录里写的原话。

**这些年长成了什么样：** 自己认定的事，谁说也没用。好处是有主心骨，不会人云亦云。坏处是撞了南墙才知道回头。被关在房子里那回，工人不敢放我出来，同学把我放出去了，我疯了一下午又自己回去被锁上——"老爸以为这招真管用，他不知道中间还有这么一出。"这里面有一种"我不服"的劲儿。

**它带来的好处：**
- 有主见，不容易被带偏
- 遇到阻力不会马上放弃
- 关键时刻扛得住压力

**它带来的麻烦：**
- 别人说话听不进去，尤其是正在兴头上的时候
- 撞了南墙才回头，代价有时候比较大
- 在合作和关系中，容易显得固执、难商量

**对照检查清单：**
- [ ] 最近有没有别人劝你但完全听不进去的事？
- [ ] 回头看，那件事上别人的意见有没有哪怕一丁点道理？
- [ ] 你是"有主见"还是"只是不想听他的"？这两个不一样。
- [ ] 如果给自己认定的事打一个"确信度"的分数（0-100），你一般打到多少才肯重新考虑？

## 二、妈妈留给我的那些东西

写妈妈的那一章读下来，有些东西看着看着就笑了——因为我身上也有。性格这个东西就是这样：你以为你是你自己，其实你身上有一半是你妈。

### 急性子

**妈妈的样子：** 妈妈是个急性子，做事情风风火火的，走路快、说话快、手上干活更快。纸箱厂上班、回家做饭、还要抽空去外婆家帮忙，一个人恨不得掰成两半用。"累肯定是累的，但她那个性子也不大会坐下来跟你说'我好累'——没那个时间，也没那个习惯。"

**我身上的样子：** 我不完全像她那么快，但骨子里那种"别磨蹭""赶紧弄完"的劲儿一模一样。事情堆在那里就不舒服，总想一口气弄完。等不了别人慢吞吞，也等不了自己慢吞吞。

**它带来的好处：**
- 行动力强，不太拖延
- 事情多的时候扛得住——妈妈一个人掰成两半用，我也差不多

**它带来的麻烦：**
- 有时候急中生乱，该想的没想清楚就动手了
- 对慢节奏的人和事缺乏耐心
- 容易替别人着急，把别人的事揽到自己身上

**对照检查清单：**
- [ ] 今天有没有因为着急而做得不够好的事？
- [ ] 那个"急"是真的有必要，还是只是习惯了快？
- [ ] 有没有因为急而对别人说了重话？

### 不服软

**妈妈的样子：** 三泰街出来的女人，从小当家当惯了。你硬我比你更硬。和爸爸吵架——老爸在部队待了十几年，说一不二、命令式的；妈妈呢，也是个不服软的性子。两个人碰在一起，谁也不让谁。

**我身上的样子：** 在科大小孩面前也是这样——心里知道打不过，嘴上不饶人。跟那个北方小伙伴打架没打赢，但第二天该一起玩还是一起玩。嘴上是输了的，态度上没输。"打不过人家"是心里知道的，但认输的话不会说出来。

**它带来的好处：**
- 不会被人欺负到低头
- 遇到打压能扛住
- 尊严感强，不会为了利益丢掉底线

**它带来的麻烦：**
- 该服软的时候也服不了——有时候服软不是认输，是策略
- 在亲密关系里容易硬碰硬——妈妈和爸爸吵了一辈子，有这个影子
- 吃了亏也不说，让人觉得你不需要帮助

**对照检查清单：**
- [ ] 最近有没有明明可以柔软一点但硬住了的时候？
- [ ] 那个"硬"是保护了自己，还是伤害了关系？
- [ ] 不服软和"死要面子"之间的那条线在哪里？

### 有事自己扛，不爱说

**妈妈的样子：** 妈妈有什么心事没地方说，也不习惯说。她身边不是没有人——外婆家一大家子人呢。但她就是不说。大姨在株洲，太远了；小姨也不大，说了也解决不了问题。没有知心的朋友，跟舅妈们关系又拧巴。那些心事，能跟谁说呢？所以她就不说了。什么都自己扛着。

**我身上的样子：** 我学了这套。从小到大，一个人的中午、一个人热饭、一个人洗碗、一个人挂着钥匙去上学、一个人蹲在摔碎的酱油碗前面——习惯了什么都自己解决。习惯了之后，向别人开口就变成了一件很难的事。

但这里有一个很关键的区别：**妈妈的"不说"是在一大家子里练出来的——她身边有人但选择不说。而我的"不说"是因为身边真的没有人——独生子女，没有兄弟姐妹，从小就是一个人的环境里练出来的。同一个性格，两个完全不同的来源，结果是一样的——什么都自己兜着。**

**它带来的好处：**
- 独立，不依赖别人
- 自己能处理很多事情
- 不会给别人添麻烦

**它带来的麻烦：**
- 有些事自己扛着扛着就扛成了习惯，忘了还有"说出来"这个选项
- 别人想帮你都不知道从哪下手——因为你从来不说你需要什么
- 重的东西压久了，自己都不知道怎么开口了

**对照检查清单：**
- [ ] 最近有没有一件事压在心上但没跟任何人说过？
- [ ] 如果说了，最坏的结果是什么？
- [ ] 如果说了，最好的结果是什么？
- [ ] 你是在保护别人，还是在保护自己？

### 心里有愧疚放不下

**妈妈的样子：** 1985年，妈妈又怀孕了。但计划生育政策正在推进，军队里比地方更严。爸爸正处在转医院工作的节骨眼上。最后妈妈妥协了，把孩子打掉了。"妈妈没跟我说过这件事。一个字都没提过。但我知道，她心里一直有一个缺口——那个没来到这世上的孩子。"

**我身上的样子：** 那只鸭子。我用石头打了它的脚，它一瘸一拐的，小鸡守在它旁边不走。"这件事到现在想起来，心里还是有愧疚的。不是因为挨了骂——好像也没人骂我——是那只鸭子一瘸一拐的样子、小鸡守在它身边的样子，还有那个蹲在旁边知道自己做错了事的小孩，一直留在心里。"

妈妈把肉省给我吃，把最好的留给我。我现在对身边人的方式，大概也是"自己少点没关系，要给就给最好的"。

**它带来的好处：**
- 有良心，不会做亏心事
- 对别人好是真心的好
- 在意的事情会在意很久

**它带来的麻烦：**
- 过去的事放不下，背着走很累
- 有时候不是你的错，你也往自己身上揽
- 对自己要求太高——"要给就给最好的"，但最好是没有止境的

**对照检查清单：**
- [ ] 有没有一件几年前的事，到现在想起来心里还不舒服？
- [ ] 那件事里，你的责任到底占几分？
- [ ] 如果换一个人在你那个位置上，他会不会也这么做？
- [ ] 你原谅自己了吗？

## 三、没有兄弟姊妹这件事

写外婆家的时候写得那么热闹——外婆家十个孩子，表哥表姐一堆，舅舅们带着打麻雀、做弹弓、点乙炔汽灯。但热闹是别人的，热闹完了我还是一个人回家。

### 这个空缺到底是什么

没有兄弟姊妹这件事，小时候不觉得少什么。有人一起玩就玩，没人玩就自己发呆。大了以后才慢慢意识到，有些东西是独生子女永远补不上的。

**有些话没有同龄人可以说。** 有兄弟姊妹的人，很多话在饭桌上就说了，吵一架打一架也就过去了。独生子女没有那个出口，所有事情都是自己兜着。妈妈也是这样——但她身边有人，只是不说。我是身边没人，更不会说。两代人，一样的结局。

**争夺、分享、妥协——这些功课是缺课的。** 在科大院子里跟小伙伴玩、在楼栋间打架、被弹弓铁丝打疼了只能躲——这些算是补了一点课。但真正的兄弟姊妹之间的那种"抢了你的又给你""吵了你的又帮你"，那种天然的、不用讲道理的情感训练，我没经历过。所以在人际关系里，有些东西反应会慢半拍——不太会争，也不太会让，界限感要么太强要么太弱。

**"一个人"的底子打得太深了。** 从小到大，一个人的中午、一个人热饭、一个人洗碗、一个人挂着钥匙去上学、一个人蹲在摔碎的酱油碗前面——独处的能力很强，但太强了也不是好事。习惯了什么都自己解决之后，向别人开口就变成了一件很难的事。自己待着最安全，但久了就不知道怎么出去了。

### 这个空缺怎么面对

缺口补不上。没有兄弟姐妹这件事不会因为你想要就变出来。但有一个东西可以做：**把"归属感"从"地方"转移到"人"身上。**

从小习惯了找出口而不是找落脚点，因为在每个地方都待不长久。但现在不一样了——不再是那个随时要搬家的小孩。可以选择在一个人、一段关系里多待一会儿。归属感不是等来的，是待出来的。没有兄弟姐妹，但朋友可以慢慢变成那种"不用讲道理"的关系。只是需要时间，而且需要自己主动留下来。

不是要填补那个空缺——空缺是补不上的。但是可以在空缺旁边建一些新的东西。

### 对照检查清单

- [ ] 最近有没有觉得"如果有个兄弟姐妹就好了"的时刻？
- [ ] 在那个时刻，你具体需要的是什么？（是倾诉？是帮忙？还是仅仅是有个人在？）
- [ ] 那个需要，有没有可能从朋友、伴侣或者其他关系里得到？
- [ ] 你有多久没有主动联系过一个真心朋友了？

## 四、那时候在长沙跑来跑去的小孩

现在回头看，我小时候那种跑法——从西牌楼跑到三泰街，从三泰街跑到科大，从科大南门跑到北门，从坡上跑到坡下，从荒地跑到水泥管子、防空洞、靶场、游泳池——像我现在这个年纪的人，在长沙有这样经历的应该不多了。

### 那种跑法的本质

那时候的小孩普遍都在外面跑。但我那个跑法有不一样的地方——**我是没有归属地地跑。**

西牌楼的小黑屋太小了，待不住。三泰街的外婆家热闹，但那不是我的家。科大的院子大，但我是从外面插进来的。每个地方都在换，每个地方都只待了几年。所以我不是在一个地方跑来跑去——我是在不同的地方之间跑。

### 它留下了什么

**不太怕换环境。** 到一个新地方不慌，因为经历过很多次了。这是好事。适应能力强，到哪都能活。

**但也不太容易对某个地方产生归属感。** 到一个新地方，第一反应不是"我怎么在这里扎根"，而是"如果哪天要走了，从哪里出去"。这不是故意的——是小时候训练出来的本能。

**一直想找一个"自己的地方"。** 从盯着那个木头房子模型想"什么时候我们能住进这样的房子里"，到后来一直在折腾——这个矛盾一直都在。一方面渴望扎根，一方面永远在找下一个坡去滑。

小时候读过一篇郑渊洁的童话，不记得是《狗家族》还是《黑黑》了。故事说一只小狗（或者小黑熊），在自己的小窝里有吃有喝，舒服得很，觉得自己的小天地就是全世界最幸福的地方。但是外面的世界完全不是那么回事——外面的那一套，就是文革的那一套。小狗不愿意出去，也不理解为什么要出去，就觉得窝里好。

这个画面一直留着。**一个小东西守着自己的一小块幸福，外面的世界再热闹再混乱，跟它没关系。**

后来回想起来，我自己不也是这样吗？在那个7平米的小黑屋里，在科大荒地上的水泥管子里，在防空洞里——在自己搭起来的小世界里，都好好的。出去就不一定了。"一直想找一个自己的地方"和"永远在找下一个坡去滑"——这两个看起来是反的，其实是一回事。都是在找一个安全的、属于自己的窝。只是一个找的是地方，一个找的是感觉。郑渊洁那只小狗不想出窝，我在长沙跑来跑去，说到底，也是在找一个窝。

### 对照检查清单

- [ ] 现在待的地方（工作的、生活的、关系的），你的第一反应是"找出口"还是"找落脚点"？
- [ ] 如果明天就要离开这个地方（或这段关系），你最舍不得的是什么？
- [ ] 你有多久没有在一个地方（或者一段关系里）连续待上三年以上了？
- [ ] 那个"想走"的念头，是真的有危险要走了，还是习惯性的？

## 五、各条线是怎么缠在一起的

上面这些不是孤立的。它们绞在一起，互相影响。

**好奇心和嘴欠是一条线。** 看见了想说，说了就不该说。好奇心本身是好的，嘴欠是它的副作用。所以对策不是收好奇心，是管住嘴。

**敏感和有事自己扛是一条线。** 什么都感觉到了，但什么都不说。这其实是对自己最残忍的组合——明明感觉到了那么多，偏偏不给它出口。敏感的出口是什么？是表达。但"有事自己扛"堵住了这个出口。

**主观意愿强和不服软是一条线。** 自己认定的事谁说也没用，而且越压越硬。这个在逆境里是救命的东西，在顺境里得罪人的时候也多。

**没有归属感和独处能力强是一条线。** 习惯了没有落脚点，所以干脆练就了在哪都能活的功夫。但反过来，越能独处就越不需要别人，越不需要别人就越难建立归属感——这是一个循环，而且是一个自我强化的循环。

**心里有愧疚放不下和把最好的留给别人是一条线。** 对自己苛刻，对别人大方。这里面有一种"补偿心理"——对自己不够好，所以要对别人好。但长期来看，对自己苛刻的人，对别人的好也带着某种紧绷。

### 最重要的一个关系：你和妈妈

所有的这些线，最后都指向一个人——妈妈。

- 急性子像她
- 不服软像她
- 有事自己扛像她
- 有了好的先给别人像她
- 心里有事不说像她
- 累了自己知道但不喊疼像她

性格这个东西，你以为是你自己的，其实翻来覆去就是小时候看着那个人的样子学的。妈妈的这一套，在她那个环境里是生存策略——一个人带娃、一个人在纸箱厂上班、一个人撑起一个家、一个人处理娘家的烂摊子。她没得选。

你有得选。

你不是必须像她那样把所有事都自己扛着。你已经不是那个帮不上忙的小孩子了。而且——妈妈知道你在写她，知道你在记白糖汽水、白辣椒拌饭、7平米的小屋。她这辈子没有亲口跟你说过那些事，但你写出来了。这本身就是一种"不说"的打破。

## 六、往后的方向

优势和劣势其实是一体两面。同一个性格，放对地方就是本事，放错地方就是毛病。所以不是要改掉自己，是在每个性格上找平衡的点。

### 可以继续喂的东西

**好奇心**是最好的家底。能看得到别人看不到的东西，这一辈子就不会无聊。不用加强，它一直在；也不用收，收了就亏了。需要留意的是给它找一个好出口——就像你写回忆录这样。有了出口，好奇心的副作用（嘴欠）自然就小了。

**敏感**是写作的本钱。不需要再加强了，已经很够用了。敏感的人疼得比别人久，但也甜得比别人真。关键是要给敏感找一个出口——写出来、说出来、画出来。堵在心里的敏感会变成内耗，流出来的敏感才是天赋。

**不服软**是骨子里的。不要收。你这一路走来，遇到坎能扛过去靠的就是这个。但可以加一个判断：分清什么时候该硬、什么时候该软。服软不是认输，是策略。

### 需要加缓冲的东西

**嘴欠加主观意愿强**——说话做事之前，给自己一个半分钟的停顿。问自己一句：**"我非说不可吗？我非现在做不可吗？"** 不是要改主意，只是给"万一"留一道门缝。这个半分钟，可能省掉很多后面要花几倍时间去弥补的事。

这个习惯怎么练？从小事开始。早上出门前，停半分钟再走。拿起手机发消息前，停半秒钟再看一眼内容。要开口说一句带情绪的话之前，咽一口唾沫。不是要你变成另一个人，只是加一个小小的减速带。

**没有归属感的惯性**——到了一个地方，认识了一些人，第一反应如果又是"出口在哪"，意识到了就告诉自己：**不急着找出口，先待一会儿看看。** 归属感不是等来的，是"待"出来的。

这个也从小事练起：同一个理发店多去几次，同一个摊子多买几次，同一个群多聊几句。不是要你强行扎根，只是给"留下来"一个机会。

### 最值得试着突破的东西

**有事自己扛**——这是妈妈留给你的，也是你最应该试着突破的一项。不是独立不好，是你已经独立了几十年了，可以试着学另一套了。不是所有事都要往外倒，但真正重的东西，试一次，说出来。天不会塌。

可以这样开始：
1. 找一个人——信得过的、不会judge你的人。一个就行。
2. 试一件小事——不是什么惊天秘密，就是一件平时你觉得自己应该消化掉、但确实有点压着的事。
3. 看看结果——最坏的情况是什么？是不是比你一个人扛着更糟？

试过一次之后你会发现：原来我说出来了，那个人没有看不起我，没有嫌我烦，也没有把我的事到处说。那下一次，就可以再说一点。

### 关于那个空缺

没有兄弟姊妹这件事，缺口补不上，但可以在缺口旁边建一些新的东西。归属感可以从"地方"转移到"人"身上。没有兄弟姐妹，不代表不能有"不用讲道理"的关系——那种可以吵架但知道不会散的关系。只是需要时间，而且需要你主动留下来。

底子已经打好了。剩下的不是改掉旧的自己，是把这套底子用对地方。速查表见[[#速查表|文首]]。

## 七、这篇回忆录放在更大的画布里

> ⚠️ 本节含AI视角——基于训练数据中的阅读经验所做的外部定位。与前面六节（基于回忆录数据的自我诊断）性质不同，读时注意区分。

写完自己的故事之后，回头看看同时代其他人的回忆——不是要比个高下，是想知道自己写的东西在更大的画布里大概站在什么位置。

### 真的回忆录长什么样

我看过不少个人的自述和回忆录（在我的训练数据里）。那些真正真实、不胡编乱造的，有几个共同特征：

**第一，写丑事。** 真的回忆录不会只写光辉的。写过脸红的事、后悔的事、现在想起来还恨不得钻地缝的事。你写了拉在床上、舔味精、和狗分零食、光着身子跑回公厕——这些东西编不出来，也犯不着编。有这个，文章就立住了。

**第二，不把父母写成圣人。** 最假的回忆录里父母都是完美的、奉献了一切的、没有任何毛病的。最真的回忆录里父母是复杂的——你写爸爸"不敢抱你"又"关你禁闭"、写妈妈"急性子""不会委婉""有事不说"。但你在最后写"这篇东西，是献给他们两个人的"。这才是真实的分寸：看到了全部，依然尊重。

**第三，时空感具体。** 真的回忆录一定有具体的街道名称、房子结构、气候、声音、气味。西牌楼、三泰街、五一大道、铁路桥、石板路、咯吱咯吱的木楼梯、乙炔汽灯的嘶嘶声、火车"况且况且"过桥的声音——这些细节是装不出来的。一个人编故事的时候，会忽略气味和声音；只有真实记忆才会带着这些。

**第四，不强行升华。** 最假的回忆录每个故事后面都要加一句"这件事让我懂得了人生的道理"。你没有这样做。但你偶尔做的时候是真诚的——"走廊那头是厕所，走廊这头是害怕。中间隔着一盏坏掉的灯泡"——这不是硬加的，是事情本身就长在那个比喻里。

读过的那些真实回忆录里，有几个印象特别深的：

- **《平如美棠》饶平如**——九十岁开始画漫画记妻子，吵架、摔东西、被批斗都画出来了，不做任何修饰。风格和你有点像：不评判、不升华，只是让事情自己说话。
- **野夫《乡关何处》**——写家族命运，最难得的是他不把自己写成好人。很多回忆录作者会下意识地把自己的形象修饰得体面一点，野夫不干这个事。这一点你也是。
- **杨显惠《夹边沟记事》**——口述史式的记录，把说话的权利完全交给当事人，不加自己的判断。你写妈妈打掉孩子那段也做了类似的事：只是陈述，没有说谁对谁错，让读者自己感受。

你的回忆录往前走了一步：你不仅写了发生了什么，还回过头去理解——理解爸爸为什么暴躁、妈妈为什么不说。这一步很多人走不到，不是因为能力不够，是因为原谅比记录难。

### 同时代人，完全不同的童年

同一个年代（1970-80年代）的中国，不同的人过的几乎是不同的世界。

**城里孩子 vs. 乡下孩子。** 你写外婆家"那时候拉肚子也能死人"——这句话，城里孩子和乡下孩子的感受完全不同。1960-70年代出生的农村孩子，记忆里的关键词是：饿、干活、交不起学费、没有电、没有自来水、从小就要挣工分。你爸爸的自述里写的那些——三岁吃不上饭、领赈灾粥、抓鱼交学费——那是真正农村孩子的常态。你在西牌楼和科大虽然也苦，但至少是城里的苦——有纸箱厂上班的妈妈、有军人工资支撑的爸爸、有自来水有电有电视。差在一个"还有没有退路"。城里孩子再苦，家里还有工厂、有单位、有邻居；农村孩子那种穷是真的没有退路——你爸爸写他当通讯员之前，家里连三块钱的学费都拿不出来。

**大院子弟 vs. 街头孩子。** 这一点你在回忆录里自己就写了——"科大小孩的条件比街上孩子好太多了"。大院子弟（军队大院、高校大院）是中国一个很特殊的群体。他们不出大院门就能解决一切：幼儿园、小学、游泳池、电影院、服务社、靶场、广播站。街头孩子没有这个封闭的、自给自足的小世界。科大的围墙既是物理边界也是心理边界——墙外是社会上的乱，墙里是你的前坪、荒地、水泥管子。你的童年有层保护，街头孩子没有。

**独生子女 vs. 有兄弟姐妹。** 你夹在中间——外婆家十个孩子的热闹你看见了，但热闹完了你是一个人回家。比你大十岁的人（像你的舅舅们），记忆里是被兄弟姐妹"夹着"长大的——衣服是老大穿完老二穿、饭是抢着吃的、架是跟亲兄弟打的。比你小十岁的人，连"有兄弟姐妹是什么感觉"这个概念都没有。你是那个站在门槛上的人——知道热闹是什么样子，但热闹是别人的。

**同一座城市，不同的层次。** 你的回忆录里其实有好几层长沙：西牌楼的石板路老长沙、三泰街的木楼作坊长沙、科大的体制内单位人长沙。这三层在1979-1985年间同时存在，彼此只隔几百米，但里面的孩子过着完全不同的日子。大多数人只活在其中的一层里。你能同时看到这三层，是你的出身和经历给你的一个独特视角。

### 一个提醒

同时代人的回忆录，多数会有一个倾向：要么把过去写得太苦（"那时候真不容易"），要么把过去写得太好（"那时候真单纯"）。你两样都没做。

你写苦——7平米、药罐子、关禁闭憋尿、蛔虫——但没有抱怨。你写乐——白糖汽水、萤火虫、滚铁圈——但没有美化。这个"不偏不倚"不是文笔问题，是对自己诚实的结果。

还有一点：你写的是"一个具体的小孩的经历"，而不是"一个时代的缩影"。很多回忆录作者会不自觉地把自己的经历拔高成"我们这一代人如何如何"，但你没有。你一直写的是"我"——我吃的苦、我看到的、我记住的。这个谦虚，让文章有了真实的重量。
