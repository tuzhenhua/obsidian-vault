# 交易日志模板

> 每次买卖填一份。清空感情，只看逻辑。

---

## 买入记录

| 项目 | 内容 |
|------|------|
| 日期 | |
| 股票名称/代码 | |
| 买入价格 | |
| 买入了多少（仓位） | |
| 买入前通过了选股检查清单吗？ | 是/否，哪条没过 |
| **用一句话说：为什么买？** | |
| 预期持有多久？ | 短线(<3月) / 中线(3-12月) / 长线(>1年) |
| 什么情况下会卖？ | (提前写清楚——业绩变差/估值过高/基本面变坏/找到更好的) |
| 当时大盘什么情况？ | (疯涨/正常/恐慌) |
