# 投资框架总览

> 一套股票选股和操作的决策框架。不是教你买什么，是帮你每次动手之前有一套固定的思考流程。
> 用熟了以后，像回忆录的校验系统一样——信号筛出来，噪声挡外面。

---

## 框架结构

| 文件 | 干什么 | 频率 |
|------|--------|------|
| [[选股检查清单]] | 看中一只股票 → 逐项过一遍 | 每次选股前 |
| [[公司分析模板]] | 深入分析一家公司 | 选股通过初筛后 |
| [[交易日志模板]] | 每次买/卖 → 记录决策理由 | 每次交易 |
| [[错误清单]] | 亏钱的教训 → 归类存档 | 每笔亏损后 |
| [[核心概念速查]] | PE、ROE、现金流…一句话讲清楚 | 随时查 |

---

## 决策流程

```
发现一只股票
    ↓
[[选股检查清单]]  → 不通过 → 放弃，记录为什么放弃
    ↓ 通过
[[公司分析模板]]  → 深入看
    ↓
决定买入 → [[交易日志模板]]（记录：为什么买、什么价、多少仓位）
    ↓
持有期间 → 定期复查公司分析
    ↓
卖出 → [[交易日志模板]]（记录：为什么卖、盈亏、学到了什么）
    ↓
亏损 → 加入[[错误清单]]
```

---

## 三条铁律

1. **不懂不买。** 说不清楚这家公司靠什么赚钱的，不碰。
2. **每次交易写日志。** 不写的操作等于白做——你不知道自己为什么对、为什么错。
3. **错误清单每月翻一次。** 防止同一个坑踩两次。

## 学习路径

先建骨架，再填内容：

- [ ] 读完[[核心概念速查]]，把每个概念用自己的话复述一遍
- [ ] 翻翻旧[[交易投资/股票/策略与心得/经验总结（股票）|经验总结]]，把有价值的教训填进[[错误清单]]
- [ ] 找一只熟悉的股票，用[[选股检查清单]]过一遍
- [ ] 模拟一次买卖，用[[交易日志模板]]写下来
