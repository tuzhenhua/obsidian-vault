# 选股检查清单

> 看中一只股票，先过这五关。任何一关通不过，搁置或放弃。
> 这不是"好股票"的判断——是"我能看懂的好股票"的判断。

---

## 一、生意模式——它靠什么赚钱？

- [ ] 用一句话说清楚这家公司卖什么、客户是谁、为什么客户选它不选别人
- [ ] 它的钱是**一次性赚**（卖产品）还是**反复赚**（收订阅/服务费/耗材）？反复赚的优于一次性
- [ ] 客户换别人的成本高不高？换起来越麻烦越好
- [ ] 这个生意十年后还会存在吗？

> ❌ 说不清楚 = 放弃。不管涨了多少，跟你没关系。

---

## 二、财务健康——数字有没有问题？

- [ ] 连续三年**营收增长**（不是一年好两年差）
- [ ] 连续三年**净利润增长**（营收涨利润跌 = 有问题）
- [ ] 经营性现金流 > 净利润（赚的是真钱，不是纸面利润）
- [ ] 负债率不过高（和同行业比，不要比同行高出一截）
- [ ] ROE（净资产收益率）连续三年 > 10%
- [ ] 应收账款的增长速度不超过营收增长速度（客户没赖账）
- [ ] 没有大额商誉（收购买贵了的雷）

> ❌ 现金流常年低于净利润 = 放弃。账面赚钱拿不到手的公司，迟早出事。

---

## 三、护城河——凭什么别人抢不走它的生意？

- [ ] 品牌：客户会因为牌子选择它而不是便宜的替代品？
- [ ] 规模优势：做到最大之后成本比别人低？
- [ ] 网络效应：用的人越多越好用？
- [ ] 牌照/专利：法律保护，别人不能做？
- [ ] 切换成本：客户换供应商太麻烦？

> ❌ 一条护城河都没有 = 放弃。没有护城河的利润迟早被竞争对手吃掉。

---

## 四、管理层——管公司的人靠谱吗？

- [ ] 大股东有没有频繁减持（跑路信号）
- [ ] 高管薪酬和公司业绩匹配吗（亏钱还拿高薪的 = 不靠谱）
- [ ] 公司说的话和做的事一致吗（年报计划和实际动作对比）
- [ ] 有没有频繁变更主营业务（什么热做什么 = 没方向）

> ❌ 大股东连续减持 + 频繁跨界 = 放弃。

---

## 五、估值——现在这个价，贵不贵？

- [ ] PE（市盈率）跟同行业比，是偏高还是偏低？
- [ ] PE 和公司历史PE中位数比较，现在是高是低？
- [ ] 如果未来三年利润不增长，现在的价格扛得住吗？

> ❌ PE 远超同行和历史中位数 = 不是不能买，是要想清楚为什么你愿意付这个溢价。
