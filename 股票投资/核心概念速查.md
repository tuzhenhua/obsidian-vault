# 核心概念速查

> 一句话说清楚。不追求严谨定义，追求"听完就能用"。

---

## 利润表相关

| 概念 | 一句话 |
|------|--------|
| **营收** | 公司卖东西收到的所有钱 |
| **净利润** | 营收减掉所有成本、税之后真正剩下的 |
| **毛利率** | 公司产品本身的赚钱能力。高毛利 = 产品好，低毛利 = 苦力活 |
| **净利率** | 卖100块最后落袋几块。越高越好，但要和同行比 |

---

## 资产负债表相关

| 概念 | 一句话 |
|------|--------|
| **总资产** | 公司手里所有的东西 |
| **总负债** | 公司欠别人所有的钱 |
| **净资产** | 资产减负债——真正属于股东的 |
| **资产负债率** | 负债÷资产。太高 = 借钱太多有风险，太低 = 可能没用好杠杆 |
| **商誉** | 收购时付的溢价。过高 = 可能买贵了，未来减值就是雷 |

---

## 现金流相关

| 概念 | 一句话 |
|------|--------|
| **经营性现金流** | 做生意真正收到的现金。比净利润更真实——利润可以"做"，现金骗不了人 |
| **自由现金流** | 经营现金流减去维持生意的必要开支。这才是公司真正能支配的钱 |

---

## 估值相关

| 概念 | 一句话 |
|------|--------|
| **PE（市盈率）** | 股价÷每股净利润。多少年能回本。PE=20 就是 20 年 |
| **PB（市净率）** | 股价÷每股净资产。股价相对资产打了多少折/溢价 |
| **ROE（净资产收益率）** | 净利润÷净资产。公司用股东的钱赚钱的效率。长期 < 10% 的慎碰 |

---

## 其他

| 概念 | 一句话 |
|------|--------|
| **分红** | 公司把利润分给股东。分红稳定 = 公司真有利润且愿意分享 |
| **股息率** | 每股分红÷股价。存银行的替代指标 |
| **ETF** | 一篮子股票打包卖。买一个ETF = 同时买了几十上百家公司 |
