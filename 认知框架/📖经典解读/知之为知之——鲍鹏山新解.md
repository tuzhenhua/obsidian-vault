# 知之为知之——鲍鹏山新解

> 《论语·为政》：知之为知之，不知为不知，是知也。

---

## 传统解读：诚实面对无知

知道就是知道，不知道就是不知道——别装。这是道德品质层面的解读：人对自己的无知要有诚实。

## 鲍鹏山的解读：从"诚实"翻成"策略"

鲍鹏山把句式重新拆解了。第一个"知"不是名词，是**动词**——追求。

- **"知之为知之"**：应当知道的，就去追求、去实行
- **"不知为不知"**：不应当知道的，就不必追求
- **"是知也"**：能分辨什么该追、什么不该追，这才叫真正的智慧

这个翻转让同一句话从"面对知识的态度"变成了"面对信息的生存策略"。庄子"吾生也有涯，而知也无涯"是同一个意思——以有限追无限，不筛选就淹死了。

| | 传统 | 鲍鹏山 |
|---|---|---|
| **焦点** | 认知态度（求真） | 知识策略（求效） |
| **目标** | 不装懂 | 不被无用信息淹没 |
| **问题** | 你是不是在骗人 | 你是不是在浪费注意力 |

## 两个层级：筛选 vs 不玩

鲍鹏山停在"选择性学习"——在场内做减法。但还有更前一步：**不进入那个场。**

- **筛选**：信息进来了，再判断要不要。费精力，且判断本身有成本
- **不玩**：直接关掉那个信息源。不是"这个公众号要不要取关"，是"这个平台我根本不用"

关掉输入比过滤输入省力一百倍。今天删了四篇文章不是筛选——是直接不犹豫、不存档、不"万一以后有用"。脑子不会爆炸，不会长白头发，不会愁眉苦脸。

## 一个隐含风险：标准从哪来

"不知为不知"的前提是你得先知道什么"不应当知道"。如果标准错了呢？

- 标准太宽 → 什么都没挡，照样淹死
- 标准太窄 → 把该进来的也挡在外面，变成偏见放大器

所以过滤不是一锤子买卖。过滤完之后，还得用[[信号穿过噪声——我的内部校验系统|校验系统]]回头对答案：我是不是把不该挡的也挡了？是不是漏了什么应该知道的？过滤给规则，校验给修正——这和其他基石是同一条线。

## 和你框架的完整流程

批判性思维那篇讲的是**"放→收→融合"**的节奏。这篇补了前面一步：

```
不玩 → 过滤 → 放（创造性） → 收（批判性） → 融合（行动）
 ↑                ↑                              ↑
输入端镜子      想法                            豹变
  ↑                                    
校验系统回头对答案
```

五步缺一步都不完整。敢选择"不学"难，敢选择"不玩"更难——但这是输入端的第一道关。关好了，后面四步才不白跑。
