# 《毛选》中的战略思想：动态平衡、不对称、净评估

> 一个复杂且不对称分布的世界里，怎么活下来？具备战略思想，找到动态平衡的那个点，然后一遍一遍去验证、去用，直到做成。光"持续"二字就把大多数人筛掉了。

毛泽东《毛选》中三篇核心著作——《中国革命战争的战略问题》《论持久战》《目前形势和我们的任务》——加上十六字方针，藏着动态平衡、净评估、不对称这三把分析工具。

---

## 一、战略是什么

**两层含义：思维 + 方法。** 思维定方向，方法去执行。二者缺一不可。

渔夫捕鱼：战术是"怎么捕"，战略是"捕不捕、为什么捕、捕哪类、在哪捕、什么时候捕"。大多数人只盯着捕鱼手法，但决定捕到多少鱼的，是前面那五个问题。

**战略的关键是间接路线。** 李德哈特：一个新观念如果直接进攻，必然激起顽强抵抗。渗透、侧击，反而更容易。用战术上的勤奋掩盖战略上的懒惰，是假勤奋；反过来，也是幻想。

## 二、净评估：三步分析镜

净评估听起来复杂，本质就是一个**对照分析工具**——不只看自己，是看自己和外部之间的动态关系。毛泽东的方法"去粗取精、去伪存真、由此及彼、由表及里"，就是净评估的核心。简化成三步：

### 第一步：画四个格子

面对任何一件要做的事，列四行：

| | 内部 | 外部 |
|---|---|---|
| **有利** | S 我强在哪 | O 外面有什么机会 |
| **不利** | W 我差在哪 | T 外面有什么威胁 |

写完就停，不要展开成论文。

### 第二步：看四个交叉

不看单个格子，看**它们碰在一起发生什么**：

| 交叉 | 问题 | 信号 |
|---|---|---|
| S + O | 我强的地方，外面刚好有机会？ | 有→加注。没有→优势用错了地方 |
| W + T | 我弱的地方，外面刚好在压过来？ | 有→危险区，优先补 |
| S + T | 我强的，是不是被环境削弱了？ | 是→优势在贬值，得换打法 |
| W + O | 机会来了，因为短板抓不住？ | 是→这个短板优先修 |

### 第三步：定一个动作

扫一遍四个交叉，挑**最烫手的一个**，只干一件事，别想全搞定：

- S+O 最亮 → 加码，集中兵力打
- W+T 最烫 → 先修短板，别急着扩张
- S+T 突出 → 换环境或换打法
- W+O 最急 → 专门补这个短板，机会不等人

和[[探针系统——碎片平台测试方法论|探针]]的逻辑一样——净评估给快照（"我现在大概站在这"），探针扔碎片去验证（"真是这样吗"）。快照错了就调，和打卦校准爻位一个道理。

## 三、六步法：怎么研究一个问题

毛泽东在《中国革命战争的战略问题》里示范了一套方法。他研究战争的路径：**战争 → 革命战争 → 中国革命战争**——从一般到特殊，从特殊到更特殊。每一步都在缩小范围，逼近具体。

对应到个人，同样的递进：**奋斗 → 个人奋斗 → 人生奋斗。** 不是喊口号，是不断限定——你到底在跟什么斗、在哪斗、什么条件下斗。

操作上六步：

1. **定对象**：你要做成什么事？不把对象搞清楚，往下都是空的
2. **找规律**：这件事的性质是什么？它和外面的事有什么关系？抓不住规律就做不成
3. **明目的**：做这件事到底为了什么？毛泽东研究战争是为了消灭战争
4. **看全局**：这件事的全局规律是什么？有没有全局视角
5. **真学习**：读书是学习，使用更是学习，从做中学
6. **定战术**：十六字诀"敌进我退、敌驻我扰、敌疲我打、敌退我追"——敌强时不硬碰，积攒实力；有实力了抓主要矛盾打；打完不松懈，下一波问题还会来

毛泽东说：**我是靠总结经验吃饭的。** 真正的理论只有一种——从客观实际中抽出来、又在实践中验证过的。没经过检验的，不算。

## 四、《论持久战》：动态平衡的实战

1938 年，抗战十个月后。毛泽东做了四件事：

**先定基调。** 全部问题的根据：中日战争是半殖民地中国和帝国主义日本之间在 20 世纪 30 年代的决死战。不是什么抽象战争，是**这个**战争。

**再拆特点。** 敌强我弱、敌退步我进步、敌小我大、敌寡助我多助——四个特点画出不对称格局。

**然后分阶段。** 防御 → 相持 → 反攻，三个阶段各打各的。

**最后配工具。** 不同阶段用不同打法：内线防御 + 外线进攻、正面运动战 + 敌后游击战。持久战是战略概念，游击战和歼灭战是战术概念——层级不能乱放。

核心方法：**内外、正反、全局局部，多维度同时看。** 不是找一个完美答案，是在多个维度之间找动态平衡点。

## 五、十大军事原则：行动路线

出自《目前的形式和我们的任务》。每一条都是"怎么打"的具体回答：

| 原则 | 精髓 | 对个人的意思 |
|---|---|---|
| 先打分散孤立之敌 | 柿子捡软的捏 | 别一上来就啃最难的事，先从小事做起，做极致，积累杠杆 |
| 先取小城市再取大 | 农村包围城市 | 选能力圈内的开始，小→中→大，别眼高手低 |
| 歼灭有生力量 | 只打有意义的 | 只做对全局目标有推进的事，别做看起来很忙的废动作 |
| 集中绝对优势兵力 | 集中力量打一点 | 发扬优势到极致，优势会掩盖短板。追求完美的人最没用 |
| 不打无准备之仗 | 准备决定结果 | 积累，再积累。看起来毫不费力的人，准备做足了 |
| 不怕疲劳连续作战 | 扛住就是优势 | 面对困难，有人逃有人扛。人生差别就在这里 |
| 运动战+阵地战 | 灵活换打法 | 坚持正确的事，但方法要对。方法错了，努力白费 |
| 夺取守备薄弱的据点 | 攻击弱点 | 把主要精力放在困难最薄弱的突破口，一次性解决，不让反复 |
| 人力物力来源在前线 | 资源从离问题最近的地方来 | 别闭门造车——真问题、真反馈都在一线，不在书斋里 |
| 两战役之间休整 | 节奏比冲刺重要 | 会休息才会打仗。身体是一切的本钱 |

## 六、收束

毛泽东的战略思想，落到最后是一句话：**把敌人搞得少少的，把朋友搞得多多的。**

它不是让你去打谁——是让你在任何复杂局面里，知道敌人在哪、朋友在哪、自己站在哪。然后选对路线，持续打。动态平衡不是一次算出来的，是一次一次打出来的。
