# 逛——框架的软模式

> "在任何领域都可以去摸摸看看，只要不掉进去，还蛮有意思。"
> ——2026-07-01

---

## 逛 vs 翻

框架之前只有"翻"——带目标进去，要翻出东西来，翻不完会卡住。
现在多了"逛"——空着手进去，不一定要翻出东西，逛不完就走了。

| 翻 | 逛 |
|------|------|
| 带目标进去 | 空着手进去 |
| 要翻出东西来 | 不一定要有结果 |
| 翻不完会卡住 | 逛不完就走了 |
| 容易变成死磕 | 不会变成藤蔓 |
| 容易掉进去 | 不掉进去 |

逛是翻的前置动作——先在浅水区逛逛，觉得有意思再往深了走。不想翻的时候就逛逛，不追数据、不追目标、不追自己。

## 背面

目标清晰 → 死路一条。
表面上有方向的东西都有问题——因为那个方向大概率是别人画好的。
反过来 → 退一步海阔天空。不是后退，是撤销被画好的方向，退回所有方向都还开着的状态。
