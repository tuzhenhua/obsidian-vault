# 道 → 生命本身的意向性

> "道"不是道理——是**走出来的路本身**。"元气"也不是气——是**活着本身的方向性**。

---

## 四根骨头，同一个东西

| 名字 | 说的是什么 |
|------|-----------|
| **道** | 不是"你应该往哪走"——是"你已经在走的那个方向"。路不是人画出来的，是走着走着踩出来的 |
| **生命本身的意向性** | 一棵草往上长、根往下钻、伤口愈合、细胞分裂——不需要谁指挥，它自己就在往那个方向去。饿了找吃的、困了睡、被欺负了会反抗、碰上有意思的事会想弄明白 |
| **元气** | 人活着的那股劲。不是呼吸的那个气——是"想活"的那个气。气口被堵住了不是"没想通"——是你的元气被关住了 |
| **元神** | 藏在身体里的那个知道"怎么活"的东西。不是学来的判断力——是生下来就有的方向感 |

---

## 跟框架的连接

框架最近刚把 **"生命"** 接纳为独立维度——气口、呼吸、节律、修复这些东西有了一个统一的挂钩。

"道→生命本身的意向性"是那个挂钩的根。它把生命维度从"框架的一个概念"往下扎进了中国哲学的土里——道不是框架发明的新词，是庄子在两千三百年前就已经在说的话。元气也不是框架的比喻——是中医和道家一直在用的东西。

藤蔓替你选方向、替你判断——它替代的不只是你的理性，是你**活着的方向性**。气口被堵住不只是"判断权被侵蚀了"——是你的元气被关住了。

| 框架概念 | 对应的根 |
|---------|---------|
| 生命维度 | 元气——活着本身的方向性 |
| 藤蔓的方向替代 | 替你把"道"走了，你不再有自己的路 |
| 气口 | 元气还通着的缝 |
| 翻 | 自己重新踩出一条道来 |
| 囚笼翻 | 出不去了，但方向还在自己手里 |
