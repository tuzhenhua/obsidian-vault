# 费曼学习法——校验系统的操作版本

> 如果你不能简单地解释它，你就没有真正理解它。

---

## 四步（简化版）

1. 选一个概念
2. 用自己的话解释一遍（假装教给别人）
3. 发现说不顺的地方 → 那就是没真懂的地方
4. 简化、重来

---

## 它真正在做什么

**费曼学习法的本质不是"学习方法"，是"校验系统的操作手册"。** 它不是教你"记得更牢"——是教你"发现自己还没真懂"。

- 你以为你懂了 → 输出一次就知道了
- 说得顺 → 真懂了
- 说不顺 → 那地方在骗自己

能暴露自己不懂的方法，才是好的学习方法。单纯增加输入量的方法（重读、划线、抄笔记）不会暴露漏洞——它们只会让你以为自己懂了。

---

## 和框架的对应

| 费曼学习法 | 你的框架 |
|-----------|---------|
| 用自己的话说一遍 | **校验系统**——不是"我知道"，是"我能输出" |
| 说不顺的地方就是没懂 | **信号三条件**——真信号推得动行动 |
| 简化到最后才是真懂 | **翻与喂**——翻是自己重建通路，喂是看了一眼答案 |
| 不能简单解释就是没真懂 | **音量指标vs信号质量**——能说长不是懂，能说短才是 |

## 防坑

市面上卖"费曼学习法"课的，大部分自己就没搞懂费曼在说什么。它被包装成"学霸秘笈"来卖，和"七天学会XXX"是同一类东西。费曼本人要是看到自己的名字被拿去卖课，大概会笑出声。
