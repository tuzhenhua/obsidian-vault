普通人努力、奋斗、拼命，却原地打转；

高手举重若轻，就可以实现目的；

为什么？因为他们懂人性？

不完全是！

有一个词，叫“小天地”，我说的不是你的那间书房。

小说中有一个神奇的场景：

在自己的“小天地”中，战力至少提升一级，敌人如果是同一个境界的话，根本不够打。

那这个“小天地”是什么？

以前理解得很泛，以为就是主场的意思。

你是主，来者是客，客随主便，当然会遵循你的规则来。

就好比你去别人家做客，在没有得到主人的允许下，一定不会随意进入卧室。

这就是主场的优势！

而高手，就有这么一座“小天地”！

  

可是在现实生活中，普通人如何构建自己的“小天地”呢？

除了居家待客这个浅显的例子之外，实际生活中，那又如何才能让自己“战力”提升，做事情有如神助呢？

破解《阴符经》过程中，我发现了一个词：结构！

结构，不是单纯意义上的组织结构，不是谁向谁汇报，也不是谁给谁安排工作。

结构，是河床，不是河水，它是有力量的！

在结构中，“定义”了一系列的行为集合。而别人只要在这个结构中，虽然他们的行为有所不同，但只要在这个行为集合中，那结果就不会偏离太多。

更不会因为人变了，结果就全然改变。

你也可以理解为“容器”，不同场景的结构，就好比不同的容器。液体倒入其中，在这个地球上，不论什么液体，但只要是液体，最终定型成容器的形状。

如果你能构建结构，你就是高手！

你就可以不知不觉的将他人拉入到你的结构中，当然事半功倍。

但是，在你能够构建自己的结构之前，你首先要做的，是破局。

破局破多了，自然就会构建自己的结构“小天地”。

你有没有想过，我们自己，本身就身处不同的结构中，有些是人为构建，有些是自然形成。

那什么是结构？

结构，是一个系统中，各个要素之间固定的链接方式、运行规则和反馈回路。

说人话就是：你身在一个“局”里，这个局有它自己的剧本。你以为是你在做选择，其实是剧本在替你走剧情。

说两个最日常的例子，把结构的零件拆开给你看。

第一个例子：减肥反弹

要素：你的身体、节食行为、饥饿感、进食行为、代谢率。

链接：节食 → 饥饿感上升 ；饥饿感上升→ 进食冲动增加；节食 → 代谢率下降。

反馈回路：节食 → 代谢下降 → 恢复正常吃饭后体重反弹更快 → 更强烈的挫败感 → 更极端的节食。这是一个恶性增强的回路。

规则：身体的默认规则是：身体能量短缺的时候，会降低消耗、加强储存。

杠杆点：不是“更拼命地节食”，而是改变进食顺序（比如先吃菜肉，后吃主食）或增加蛋白质，让血糖平稳、维持代谢水平不降，也就是要打破节食→代谢率下降的链接；

所以很多人的减肥失败，根本不是不自律，而是被“节食-代谢降低-报复性进食”这个回路绑架了。

  

第二个例子：恋人之间的冷战

要素：你、对方、焦虑的情绪、沉默的行为、攻击性的语言。

链接：你的焦虑 → 你提高音量/质问；对方感到被攻击 → 对方沉默；对方沉默→ 你的焦虑升级。

反馈回路：焦虑 → 追问→ 沉默 → 更焦虑 → 追问/质问→ 更沉默。这也是一个恶性增强回路，双方都被卷入。

规则：潜规则是“谁先服软谁就输了”；或者“沉默等于冷静，表达等于失控”。

杠杆点：不是“追得更紧”或“彻底冷战”，而是在情绪爆发前说一句：“我现在很难过，但我更怕你离开，我们可以停十分钟吗？”——这一句话，彻底打断了“攻击-防御”的链接，打断了那条恶性的增强回路。

所以，你们之间不是不爱了，而是被“追问/质问-沉默/逃避”的结构，接管了对话。

  

我们在长大的过程中，总是在被教育：出了问题，要从自己身上找原因。

所以，我们更自律、更努力、更忍让、更讨好。

如果是自己一个人的修行，向内求总归没错。

但每个人，身处这个社会，不免与外界产生链接。

所以这句话应该扩大成：

出了问题，要从自己身处的结构中找原因（可能是自己的原因，更可能是链接的原因）。

如果仅仅是从自己去找问题，但身处的结构不改变，那就只是在同一个反馈回路里跑得更快，磨损得更厉害。

就像一个被关在旋转门里的人，拼命跑，以为自己在前进了，其实一直在门里打转。

所以，如果连自身所处的结构都看不透，破不开，那又如何构建自己的结构，也就是“小天地”呢？

如何破开身处的结构，其实就是三招：

改变/增加链接，改变反馈回路，引入新的规则。

所以真正的高手，他们从不正面硬刚，使用蛮力，他们就是看清了结构，找到其中的杠杆点，轻轻一撬，就能够破局。

后续每一篇文章，我都会带你，也是带自己，反复的做同一件事：

画出你生活里的结构，然后找到那个最小的杠杆点。

每一篇，都会拆解一个我们每天都在面对，却从未用“结构”视角看过的场景。

我会按照个人 → 关系 → 社会 → 日常四个板块，当然也更希望你们能够有新的场景建议，我们一起剖析；

希望我们不只会“再努力一点”；

更是要看见那个隐藏的人生河床，并找到那个正确位置；

然后只挖一锹土，人生长河的水，便自会改道!
