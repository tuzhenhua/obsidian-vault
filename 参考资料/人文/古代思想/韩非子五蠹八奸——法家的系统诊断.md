# 韩非子·五蠹八奸——法家的系统诊断

## 五蠹：五大系统蛀虫

韩非子认为五类人像蛀虫一样危害国家，必须铲除：

| 蛀虫 | 原文指向 | 现代映射 |
|------|---------|---------|
| **学者**（儒墨之士） | 空谈仁义，干扰法治 | 脱离实操的"理论派"、只说不做的顾问型 |
| **言谈者**（纵横家） | 编造谎言，借外势谋私利 | 靠信息差和话术牟利的中间人 |
| **带剑者**（游侠刺客） | 聚集门徒，无视法律 | 规则之外的暴力/强权执行者 |
| **患御者**（逃兵役者） | 依附权贵，逃避国家责任 | 搭便车者、吃空饷的寄生岗位 |
| **商工之民** | 囤积居奇，不从事农耕 | 不创造价值只做分配的中间环节 |

## 八奸：八种信任腐蚀路径

韩非子列举了臣下危害君主、夺取权力的八种手段：

| 奸术 | 手段 | 信任漏洞 |
|------|------|---------|
| **同床** | 利用君主的妻妾影响决策 | 枕头风——信息通道被亲近者垄断 |
| **在旁** | 利用君主身边的侍从窃取信息 | 信息过滤——身边人筛选什么你能看到 |
| **父兄** | 利用亲属关系施压进言 | 人情绑架——血缘关系绕过制度 |
| **养殃** | 投其所好，建立情感绑定 | 恩情债——让渡信任给"对你好"的人 |
| **民萌** | 私散公财博取虚名 | 收买民心——用公器建私信 |
| **流行** | 豢养说客制造舆论 | 信号污染——当噪声被系统性地制造 |
| **威强** | 豢养亡命徒以暴力威胁 | 恐惧替代信任——暴力扭曲反馈回路 |
| **四方** | 勾结外国势力寻求外部支持 | 外部杠杆——借外力撬动内部信任结构 |

## 框架连接

- **信任的让渡与回收**——八奸就是信任被腐蚀的八条具体通道。每一条都在回答同一个问题：**"信号是怎么被过滤的？反馈是怎么被污染的？"**
- **秩序是逼出来的**——韩非子的法家思路正是"秩序靠设计不靠自发"的代表，和你那篇对"理性自负"的批判是同一主题的不同侧面
