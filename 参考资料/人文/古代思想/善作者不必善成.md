# 善作者不必善成——乐毅的功败垂成

出处：《史记·乐毅列传》/《战国策》

> 善作者不必善成，善始者不必善终。

## 两层含义

| | 含义 |
|--|------|
| **善作** | 善于做事、有本事、有能力 |
| **善成** | 把事情真正做成、取得结果 |
| **不必** | 不一定对应 |

有能力、有好的开始，只是成功的条件之一。结果还取决于时机、环境、他人配合、运气。

## 框架连接

- **善作 ≠ 善成**——做和成是两件事。善作是个人能力，善成是**系统能力**。工地上的核心矛盾就在这：墙板班组有能力安装（善作），但材料流转、工序衔接跟不上（系统不善成）。
- **赌约的验证**——"砖#3风在门外"写得很好（善作），但到不了500（不善成）。不是写得不好，是平台推流的系统条件不配合。
- **信任的让渡与回收**——"善作"是对自己能力的信任，"善成"是让系统也配合。信任需要真实反馈，但反馈回路可能被系统阻断——你有信号密度，但链路不通。
