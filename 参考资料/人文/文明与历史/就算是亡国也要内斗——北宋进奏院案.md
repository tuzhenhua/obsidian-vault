# 就算是亡国，也要内斗——北宋进奏院案与庆历新政的失败

来源：春秋小历/我是历史其实挺有趣（2026.06.04）— 3096人阅读

## 进奏院案始末

| 要素 | 内容 |
|------|------|
| **时间** | 北宋仁宗庆历四年（1044）秋 |
| **人物** | 苏舜钦（进奏院监，杜衍女婿，范仲淹举荐） |
| **事件** | 进奏院年度赛神会（联欢会），卖旧公文纸筹款办宴 |
| **导火索** | 王益柔酒后赋诗："醉卧北极遣帝扶，周公孔子驱为奴" |
| **攻击者** | 王拱辰（御史中丞）——借题打击新政派 |
| **结果** | 苏舜钦削籍为民，与会十余人贬逐出京；数月后杜衍罢相、范仲淹罢执政、韩琦罢免；庆历新政宣告失败 |

## 案件本质

> "看似整肃官风，本质是一场政治倾轧。表面上的酒食之过、狂放之语不过是借口，真正原因还是新政派触动了保守派的利益。"

王拱辰得胜后说：**"吾一举网尽矣。"**

## 核心机制：从"事"到"人"的议题转换

庆历新政的核心本来是"事"——解决三冗问题、增强国力。但党争一介入，主题就变成了"人"：

- 谁是君子？谁是小人？
- 你是不是结党？
- 一旦辩论从"该不该改革"变成"你是不是结党营私"，理性的政策讨论就消失了

> "很多好的政策，不是输给了现实上的困难，而是输给了自己人的内耗上。"

## 历史循环

北宋的悲剧在于这个模式反复重演：
- **庆历新政**（1043-1045）→ 进奏院案终结
- **熙宁变法**（1069-1085）→ 新旧党争
- **元祐更化**（1086-1093）→ 党争加剧

## 标题的深意

"就算是亡国，也要内斗"——这句话描述的是一种**集体非理性**：当派系利益凌驾于整体生存之上，组织会做出违背自身存续逻辑的选择。这正是系统动力学中"结构决定行为"的典型案例：党争的反馈回路一旦建立，系统就会被锁定在自我毁灭的路径上。
