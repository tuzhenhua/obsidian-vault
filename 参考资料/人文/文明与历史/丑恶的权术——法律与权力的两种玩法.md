# 丑恶的权术——法律与权力的两种玩法

来源：浮世撷英/春秋小历

## 核心命题

权力玩弄法律有两种经典手法：**用道德罪名当政治屠刀**（西汉），**先定罪再调查**（明朝）。手法不同，逻辑一样——法律是死的，权力是活的。

---

## 玩法一：道德是权力的面具——西汉禽兽行案

来源：浮世撷英（2026.05.07）— 1496人阅读

西汉对宗室乱伦（"禽兽行"）的惩处，同罪不同罚——有人死，有人活，全看政治需不需要杀。

### 四起案件对比

| 当事人 | 罪行 | 结局 | 政治背景 |
|--------|------|------|---------|
| 燕王刘定国 | 与庶母、弟妻、亲女奸 | **赐死，国除** | 武帝削藩，需要借口 |
| 齐厉王刘次景 | 与亲姐通 | **逼自杀，国除** | 齐国太富，中央想收 |
| 江都王刘建 | 烝父妾、奸亲妹、谋反 | **自杀，国除** | 谋反才被算账 |
| **梁王刘立** | **与亲姑母通奸数年** | **仅削五县** | 成帝朝，藩王已弱，杀之无利可图 |

### 机制

> **你乱伦不重要，你挡路才重要。**

武帝时期：藩王强大 → 需要借口 → "禽兽行"是最好用的道德屠刀 → 杀、除国、收地。
成帝时期：藩王已弱 → 杀之无利可图 → 同罪仅削五县。

谷永辩护技巧：把"禽兽行"从政治死罪矮化为"闺门之私"的道德瑕疵，再以皇家体面为由放过。

## 玩法二：先定罪，再调查——明朝翟銮案

来源：春秋小历/我是历史其实挺有趣（2026.06.13）— 919人阅读

### 案情

明朝嘉靖年间，首辅翟銮的两个儿子同时中举、同时中进士。政敌严嵩指使言官弹劾翟銮科举舞弊。皇帝下令彻查，所有考官被打入锦衣卫严刑拷问——**没有一个人认罪，没有任何证据。**

但皇帝仍然将翟銮罢相、削籍为民，两个儿子取消功名。严嵩接任首辅，把明朝政治搞得乌烟瘴气。

一年后，皇帝恢复翟銮名誉。隆庆年间正式平反，追谥"文懿"。

### 为什么明知冤枉还要治罪

> **皇帝认为翟銮"全不感惧"——不如严嵩听话、好用。**

没有忠臣和奸臣的区别，只有好用和不好用的区别。皇帝要用你，巨奸如严嵩也可以听之任之；皇帝不用你，明知道冤枉也要扫地出门。

### 金句

> "先定罪，再调查，保证没有冤假错案。"

> "半生荣辱都休问，且饮春茶看晚霞。至于你冤枉不冤枉，天知地知，你知我知，唯独皇帝，不想知。"

## 两种玩法的共同逻辑

| | 西汉 | 明朝 |
|--|------|------|
| 罪名 | 乱伦（道德） | 舞弊（程序） |
| 真相 | 无罪 | 无罪 |
| 真实理由 | 削藩需要 | 皇帝不喜欢 |
| 操纵者 | 武帝+主父偃 | 嘉靖+严嵩 |
| 你的框架 | 道德是权力面具 | 定罪是权力工具 |

同一个底层逻辑：**一切罪名都可以按政治需要随时变化解释。法律是死的，人是活的；权力是活的，道德是死的。**

## 框架连接

- **秩序是逼出来的**——法律的执行不是秩序的来源，权力才是。两案都是"道德是防举报协议"的历史注脚：道德标准不是用来信守的，是用来选择性执行的。
- **信任的让渡与回收**——当法律信号不一致（同罪不同罚，无罪也定罪），信任公式里的"信号密度"就失效了。老百姓看到的是：同样的罪，有人死有人活；没罪的也可以被定罪——信任积分必然归零。
- **左翼知识分子的认知陷阱**——"爱抽象人类不爱具体的人"。用抽象的法律/道德标准审判具体案件时，标准本身可以被权力任意弯曲。
