# 垂直分裂——东亚小孩的终身抑郁

  

## 一、什么是垂直分裂

垂直分裂（vertical splitting）是自体心理学家科胡特提出的概念。要理解它，先要理解另一种更常见的分裂——水平分裂。

水平分裂就是我们通常说的压抑。一个人的某些愿望、冲动、记忆，因为太痛苦，被压到意识水平以下。你感知不到它们，但它们会在梦里、症状里、不经意的口误里冒出来。这是一个**上下**的结构——有意识的东西在上面，被压抑的东西在下面，之间有道屏障。

垂直分裂不是上下，是左右。

在垂直分裂里，一个人同时拥有两种关于自己的认知，它们都是意识的，但彼此不连通。就像心理空间被一堵墙从中间隔开了，墙左边站着一个人，墙右边站着另一个人，两边都知道对方存在，但不承认对方说的是真的。

科胡特是在研究自恋人格的时候发现这个现象的。他观察到，一些来访者身上同时存在着两种矛盾的自体状态：

|   |   |
|---|---|
|◈ 夸大的自体<br><br>"我是特别的，我是完美的，我应该被所有人看见和崇拜。"|◈ 匮乏的自体<br><br>"我什么都不是，我有缺陷，我不值一提。"|

这两部分同时存在，而且都是意识的——这个人既知道自己的野心，也知道自己的无能感。但两者之间没有连接：当他在夸大状态里的时候，他真心觉得自己了不起；当他在匮乏状态里的时候，他又真心觉得自己一塌糊涂。他不会在觉得自己了不起的同时想起昨天还觉得自己不行，也不会在低谷时想起上周还觉得自己天下第一。

两边的情绪都是真实的，只是它们不肯见面。

在传统的精神分析模型中，自恋被看作一个需要被克服的阶段——力比多应该从自体转向客体，"健康"的标志是爱别人，爱自己反而是需要被超越的阶段。科胡特不同意这个看法。他认为自恋有自己的发展线，它应该发展成一个人的抱负、理想、自尊感。但如果这个发展被打断了，夸大自体没有被恰当地回应和整合，它就会以一种未被消化的原始形态留在人格里。

垂直分裂，就是这种**"未被消化的夸大"**的典型处境。

## 二、垂直分裂会怎样

### 1. 你永远感觉"不够"

这是垂直分裂最直接的表现。

一个18岁的来访者，在高考中取得了全省前几名的成绩。所有人都恭喜他。他坐在咨询室里说的第一句话是：

> "可我数学扣了那道不该错的选择题。"

他以为自己在谦虚。其实是那个"完美版本的我"在审判——你失去的那几分才是真实水平，得到的那些不作数。

在垂直分裂里，一个人的大脑里装着一个"完美版本的我"。这个版本是绝对的、必须达到的标准，跟"我努力能达到的目标"不是一回事。现实中真实的成就，在这个完美版本面前永远矮一截。于是产生了持续的匮乏感——已有的成绩，放在另一个参照系里永远不够。

这个感觉不局限于学业。工作中升职了——但还不够快。被领导表扬了——但他对别人也这么说。恋爱了——但她真的了解我吗？了解之后会不会走？

每一个"够到了"的时刻，那个夸大的部分立刻把标准往上提一格。

因为那个夸大的自体从来没有被现实修正过——它是最早期、最原始、最婴儿式的"我要成为全世界最重要的人"的幻想，一直原封不动地保存在人格里。

  

### 2. 夸奖落不到你身上

这是垂直分裂最微妙的表现——也是最让人困惑的。

你被表扬了。对方说了真诚的话，你也礼貌地回应了谢谢。但是那个夸奖好像被什么东西挡住了。它没有进到你的身体里，没有变成一种"我确实挺好的"的感觉。它像一颗石子扔进了一口枯井，你听到了声音，但那就是全部了。

因为夸奖的目标，是你那个夸大的部分；而听到夸奖的那个，是你匮乏的部分。它们不通。

夸大的部分在说："这点夸奖算什么？我要的是全部世界的目光。"  
匮乏的部分在说："他们不知道真实的我是什么样的，知道了就不会这么说了。"

于是夸奖悬在中间，两边都不接收。

时间长了，一个人会发展出一种微妙的行为模式：

一边疯狂地寻求认可，一边在得到认可后迅速贬值它。

不停地发朋友圈、看点赞数、在意数据——但涨了也就那样。被喜欢的时候短暂地好一点，然后很快回到基准线以下。因为外部认可只能短暂地喂养夸大的部分，但永远填不满匮乏的部分。从外界要来的确认，进不了内部那个结构性的裂口。

  

### 3. 你在"我最棒"和"我最烂"之间来回摆动

这是垂直分裂最容易被外人看到的样子。

有些人表现得很好辨认——他们看起来自大、爱炫耀、不能接受任何批评，你稍微指出一点问题，他们的反应不是愤怒就是一种"厚脸皮"的形态：夸大的部分完全外露，被否认的部分藏得很深。

但还有另一种形态，更隐蔽，也更容易被误诊：

他们看起来谦虚、低调、自我要求极高，但内心深处的秘密是——他们其实觉得自己比周围的人都强。

他们不炫耀，因为他们害怕一旦暴露了这个"我觉得自己很特别"的想法，就会被羞辱。所以他们压抑了夸大的部分，让它沉下去了。但它没有消失。它在下面，以自己的方式活着。

于是出现了一种奇特的人格配置：

**表面上**，这个人谦逊、努力、甚至自我贬低；但如果你仔细观察他的行为——他选的伴侣、他无法忍受的同事类型、他反复出现的冲突模式——你会发现一个从未被承认的信念：**他其实觉得自己应该是最特别的那一个。**

当现实不符合这个期待时，他不会直接说"我觉得自己应该比这更好"——他只会觉得痛苦、愤怒、抑郁，不知道为什么。

这就是科胡特说的"垂直分裂"的典型动态：夸大的部分没有被整合到人格中，它以原始的形态存在，要么直接冒出来（厚脸皮），要么被否认后通过症状表达出来（薄脸皮）。

  

### 4. 它的源头：一个被羞辱压垮的童年

一个孩子的夸大自体——也就是"我是好的、我是特别的、我需要被看见"——是一棵正常生长的小树。

它不需要被鼓励成参天大树，它只需要不至于被踩死。

但如果养育者不能给出那种回应——因为抑郁，因为忙碌，因为觉得"孩子不能夸"——孩子的夸大幻想就找不到出口。更严重的是，如果养育者不仅不给回应，还会在孩子的表现欲出现时予以羞辱：

"你以为自己是谁？"  
  "有什么好得意的？"  
  "这点成绩就满足了？你知道谁谁谁考了多少吗？"

那棵小树没有被踩死，但从此不再往正常的方向生长了。

它拐了一个弯。

在表面的、被驯化的那部分人格下面，原始的夸大幻想被原封不动地保存下来。科胡特称之为"被压抑的夸大自体"。它没有经历正常的现实修正——没有在"恰到好处的挫折"中慢慢调节成现实的抱负——它就那样被冰封了。

而在冰层的上面，一个人带着一个自认为"有缺陷的、不够好的、永远达不到标准"的自体，面对生活。

**这就是垂直分裂的起源。**

  

### 一个文化背景

自体心理学家David Terman在与东亚来访者工作时注意到一个让他印象深刻的现象：

太多人的成长记忆里，充斥着学业表现带来的羞辱。

那个羞辱的量级，已经超过了普通的"严格要求"的范畴。那些话反复被说：

"你让全家蒙羞"  
  "你还有脸吃饭"  
  "你怎么不去死"

当夸大自体的正常表达——"我很棒，老师表扬我了"——换来的是这样的回应时，孩子慢慢地不敢再觉得"被看见"是一件对的事。他学会的是：

我有任何想要被看见的愿望，都是错的。  
  但那个愿望没有消失。它只是被隔开了。  
  ——这就是垂直分裂的文化根源。

· · ·

垂直分裂值得被单独拿出来说，是因为它几乎不稀有——对很多在东亚教育环境中长大的人来说，它是一种常规的心理配置。

它可以被看作一个人在特定环境里长出来的生存结构。这个结构帮一个人避开了最直接的伤害——如果没有把夸大的部分隔开，就可能要反复面对"我想被看见"和"我被羞辱"之间的剧烈碰撞。隔开它，至少能活着。

但这个结构有代价。代价就是那个持续的空洞感：

得到了一些东西，但开心不起来。  
  被肯定了，但不敢相信。  
  觉得自己可以做成大事，同时觉得自己什么也做不成。

理解垂直分裂不会让它立刻消失。心理结构的改变，不是靠"知道"就能完成的。

但知道这个结构之后，一个人至少可以不把"我就是不够努力"或者"我就是太贪心了"当作对自己的解释——那些标签贴错了位置
