# 江南造船厂无围壳新型潜艇

> 2026年5月底，上海江南造船厂出现一艘造型独特的新型潜艇，卫星图像于6月1日被捕获。

---

## 基本参数

| 指标 | 数据 | 对比 |
|------|------|------|
| 长度 | ~120米 | 093型108-110米；美国弗吉尼亚级114.8米 |
| 宽度 | 10-11米 | — |
| 围壳 | **无传统指挥台围壳** | 全球大型潜艇中极其罕见 |
| 尾舵 | **X形尾舵** | 2024年首次出现在中国潜艇 |
| 推进 | 推测配备**泵喷推进器** | 替代传统螺旋桨，降低空泡噪声 |

---

## 核心特征

### 1. 无围壳设计

取消指挥台围壳是这艘艇最大的识别特征。

**取消围壳的好处：**
- 降低水下阻力和涡流噪声
- 提高航速和机动性
- 减小声学特征

**取消围壳的代价：**
- 牺牲水面瞭望能力
- 丧失极地破冰能力
- 潜望镜、通气管、电子战天线等传统安装在围壳内的设备需另作布置

### 2. X形尾舵

- 比传统十字舵机动性更好
- 舵面不突出于艇体轮廓，噪声更低

### 3. 泵喷推进器（推测）

- 093后期批次已装备，技术成熟
- 大幅降低空泡噪声

## 定位分析

### 可能性一：095型攻击核潜艇技术验证艇 ✅ 更可能

- 120米攻击核潜艇尺度
- 江南造船厂承担下一代核潜艇建造任务
- 2018年小艇 → 2025/26年大艇，典型的技术验证路径
- 葫芦岛渤海厂同期下水一艘带传统围壳的新潜艇——两条技术路线并行竞争

### 可能性二：大型无人/可选有人潜航器

- 无围壳在无人艇上更容易实现（无需考虑水面瞭望等人因工程）
- 对标美国ORCA大型无人潜航器
- 但120米的尺度远超现有无人潜航器规格

**判断：** 大概率是095型的某一种构型验证艇，与葫芦岛那艘传统围壳艇处于"并行探索、择优定型"的竞争关系。

## 战略意义

1. **设计建造能力质变**——以前一个型号反复改二十年，现在一次推两个完全不同构型的大艇同期建造
2. **时间压力增大**——多路线并行，快的先定型
3. **作战假想场景转型**——无围壳牺牲近海/极地性能，说明设计目标为**远洋深水作战**
4. **全球领先探索**——无围壳大型攻击核潜艇在全世界范围内尚无服役先例


> 这是中国潜艇建造从"追赶模仿"转向"自主探索"的一个标志性信号。

## 后续讨论：无围壳艇的作战运用与"忠诚僚机"概念

> 以下内容整理自与AI的讨论（2026-06-05）

### 1. 潜艇"忠诚僚机"的国际实践

- **美国 ORCA**：波音建造的超大型无人潜航器（XLUUV），长度约26米，已交付，设计定位为核潜艇的前出侦察/节点
- **俄罗斯 波塞冬**：核动力核弹头无人潜航器，极端路线
- 中国也在推进大型UUV项目

### 2. 无人潜艇的关键制约

水下环境与无人机有本质区别，导致全自主决策短期内不可行：

| 制约因素 | 具体问题 |
|---------|---------|
| **水下通信困难** | 电磁波在水中衰减极快，深潜时仅靠极低频或浮至潜望镜深度通信，带宽极低，无法实现无人机式的实时遥控 |
| **环境感知瓶颈** | 声呐回波识别远不如光学/雷达成熟，AI判别的可靠性和实时性存疑 |
| **打击决策红线** | 攻击决策不可能交给AI，必须有人的最终授权 |

### 3. "可选有人" VS 实战现实

**技术上**，"可选有人"（Optionally Manned）是可预见的演进方向：

> 正常巡航无人值守 → 远程下发任务规划（"到X海域巡逻Y天，重点侦察Z类目标"） → 关键节点由后方或母艇人工介入决策

**但实战层面，人的因素一时半会动不了：**

- **故障容错率极低**——水下环境封闭、维修条件苛刻，设备全无人陪，任何一个小故障（管路泄漏、电路短路、传感器漂移）都可能迅速升级为灾难性事故。有人艇上水兵可以即时处理的问题，无人艇上就是任务失败甚至失联
- **维护保障链跟不上**——潜艇是高强度使用平台，靠的是艇员日常维护保养。无人化之后，这些工作谁做？回港才维护和随时随船维护，可靠性差一个量级
- **训练文化惯性**——几十年形成的核潜艇操作训练体系、损管应急流程、指挥决策习惯，不可能因为技术可行就一夜翻篇。实战中最可信的仍然是经过长期训练磨合的艇员团队

> 技术路线可以做"少人化"的探索，但**实战操作在可预见的未来仍将以有人为主**。无围壳艇最大可能是降低编制人数，而非完全无人。

### 4. 人机协同的困境

自动化程度越高，并非越"省人"，反而对操作员提出了更苛刻的要求：

**双方都会犯错，且出错模式不同：**
- **机器的问题**：传感器漂移、软件bug、电磁干扰下的误判——故障模式往往隐蔽、非直观，人在环中很难及时发现
- **人的问题**：长时间高强度值班下的注意力下降、疲劳决策、信息过载——尤其在潜航任务中，几十天不见天日、作息紊乱，任何人的判断力都会打折
- **最怕的是"浑"的状态**——人和机器互相不确认对方在做什么，机器报了一个异常，人判断不出是真的故障还是虚警；人做了一个操作，机器不理解意图、执行了不是预期的事。这种"浑水"里一旦出事，责任归谁都是后话，关键是任务已经失败了

**越先进、越复杂，越需要人的冷静和耐心：**
- 自动化接管了常规操作，但也意味着系统状态对人的透明度下降。操作员从"亲手操控"变成了"监控者"——监控者在长时间无事可做后突然需要介入，认知切换成本极高
- 真正先进的装备，反而要求操作员有更强的**系统理解能力**和**故障预判能力**，而不是更低
- 所以少人化的前提不是"不需要那么多人"，而是"剩下的人必须更精、更稳、抗压能力更强"

> 少人化不等于简单化。**技术越先进，对艇员质量的要求反而越高，数量的压缩是有下限的。**

### 5. 人员配置变化（少人化方向）

| 岗位 | 变化方向 |
|------|---------|
| 武器控制 | 减少，AI辅助 |
| 声呐兵 | 减少，AI辅助识别 |
| **任务规划/指挥决策** | **仍需有人** |
| 轮机/损管 | 大幅减少 |
| 生活保障 | 相应减少 |

120米尺度若走**少人艇**路线，估计从传统核潜艇的~100人缩减至30-40人甚至更少。

### 6. 编队设想："有围壳带领无围壳"的狼群模式

- **葫芦岛传统围壳艇** → 指挥/控制中心，承担任务规划与前敌指挥
- **江南无围壳艇** → 前出侦察/静音猎杀，利用取消围壳带来的低阻低噪优势

> 类似空军"忠诚僚机"逻辑：**有围壳的带无围壳的，前面探路、后面指挥**，形成狼群式编队而非单纯的主打下手关系。

## 后续讨论（与 Claude Code 对话，2026-06-05）

### 1. 无围壳设计的代价可能被低估

一个更深层的问题——**潜望深度的操作空间被压缩了**。

传统围壳高度通常 5-8 米，让潜艇在潜望镜深度时，艇体本身保持在较深、较安全的位置，只有围壳和升起的光电桅杆接近水面。无围壳之后，要获得同样的传感器视野，整个艇体都必须更靠近水面——**暴露风险更高**。在反潜巡逻机活跃的海域，生存力反而可能下降。

这也反过来印证了文章对"远洋深水作战"定位的判断：在深海，这个问题不严重；在近海浅水区，这是一个真实短板。

### 2. "两条路线并行竞争"这件事本身比潜艇更有意思

这不只是"谁对谁错"的技术判断问题，而是中国潜艇工业的组织模式发生了根本变化：

| 过去 | 现在 |
|------|------|
| 一个型号反复论证十几年 | 同一需求、两种方案、同期出实物 |
| 造一两艘试验艇慢慢改 | 海上跑数据，优胜劣汰 |

背后说明两件事：
- **经费和产能不再是瓶颈**——敢这样"浪费"
- **决策层对技术路线不确定性的态度变了**——承认不知道哪个方案更好，用并行竞争来"买信息"

这在武器装备发展史上是成熟做法（美国在飞机、导弹上都大量使用），但出现在中国潜艇领域，确实是第一次。

### 3. 人机协同的"监控者悖论"（与切尔诺贝利真菌的同日讨论形成对照）

潜艇环境让自动化悖论更极端：

- 空军飞行员飞几个小时落地，疲劳积累有限
- 潜艇兵水下几十天，长期封闭、昼夜节律紊乱，认知能力本身就处于降级状态
- 自动化越高，"平时没事做"时间越长，突然被要求介入时状态越差

结论：**少人化不等于简单化，对艇员质量的要求是往上走的。**

### 4. 可进一步追问的方向

| 方向 | 问题 |
|------|------|
| 中美对比 | 美国为什么没走无围壳路线？弗吉尼亚级、哥伦比亚级都保留了传统围壳——技术判断不同还是作战场景不同？ |
| 传感器替代 | 无围壳后，原围壳内的电子战天线、通信浮标、光电桅杆如何布置？是否用艇体顶部低矮整流罩替代？ |
