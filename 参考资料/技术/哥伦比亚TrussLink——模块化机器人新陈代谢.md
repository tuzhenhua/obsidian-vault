# 哥伦比亚大学Truss Link——机器人新陈代谢

> 模块化机器人可以拆解、重组、自修复。不是"固定形态"，是按需求重新连接。
>
> 来源：Columbia Creative Machines Lab，Science Advances，2025年7月。

---

## 核心概念

**Truss Link**：一个棒状模块化单元（收缩约28cm，伸长约43cm），两端带磁性连接器。每个模块可以伸缩、爬行、磁吸连接。灵感来自生物学用约20种氨基酸构建无限蛋白质——每个Truss Link就像一个"氨基酸"。

## 验证的功能

| 能力 | 表现 |
|------|------|
| **自组装** | 模块自动组成2D形状→折叠成3D机器人 |
| **生长** | 四足机器人加装"登山杖"后下坡速度提高**66.5%** |
| **自修复** | 从1m摔碎后，模块约2分钟自动重连（92%成功率） |
| **零件更换** | 电量<15%的模块自动弹出，换入新的（模拟生物凋亡） |
| **多机器人协作** | 一个机器人可以辅助另一个完成组装 |

## 框架位置

这个项目是**开放系统的物理模型**。它展示了如果一个系统没有身份锁死，可以多灵活：

| Truss Link | 你的框架 |
|-----------|---------|
| 固定形态→按需求重组 | **本质主义→关系主义**——不是"我是什么形态"，是"我现在需要什么形态" |
| 拆掉坏模块重组 | **校验系统**——损坏了不硬撑，拆掉重连 |
| 去中心化自组织 | **边缘vs中心**——没有中央控制，每个模块自己判断 |
| 根据任务换结构 | **鲁肃的节奏**——不固守一种打法，根据环境换频率 |

**"机器人新陈代谢"本质上是在说：一个系统不能只有'执行'功能，它还得有'更换自己零件'的功能。** 身份不允许和身份默认，就是人的"零件更换系统"生锈了——旧的信念模块拆不下来，新的装不上去。

Hod Lipson原话："机器人的大脑已经飞跃了，但机器人的身体仍然是整体的、不可适应的、不可回收的。"
