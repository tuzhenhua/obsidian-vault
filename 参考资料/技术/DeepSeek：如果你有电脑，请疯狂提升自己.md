## 🔥 第一步：让电脑变成你的"第二大脑"

别再做信息的仓鼠，只囤不化。

### 📦 搭建知识库

用 **[[Obsidian]]** 或 **Notion**，把看到的有价值内容，用自己的话记下来，打上标签，双向链接。收藏夹吃灰等于零，亲手整理过的知识才是你的。

| 工具 | 是什么 | 适合场景 |
|------|--------|---------|
| **Obsidian** | 本地笔记，存你自己电脑上 ✅ **你正在用** | 个人知识库、日记、学习笔记 |
| **Notion** | 在线笔记，功能丰富 | 团队协作、项目管理、数据库 |

> [!tip] Obsidian 入门技巧
> - 用 `[[wikilink]]` 在笔记之间建立连接
> - 善用标签 `#标签名` 分类笔记
> - 安装社区插件：Templater（模板）、Dataview（查询）、Excalidraw（绘图）

### 📡 让优质信息主动找你

用 **Inoreader** 订阅顶级期刊、技术博客、行业大佬的专栏。把刷短视频的"被动投喂"，切换成 RSS 的"主动猎食"。

**什么是 RSS？**
> 一种"订阅"技术，像关注公众号一样关注网站。设置好后不用自己去翻，有新内容自动推送给你。

### 🤖 学会向 AI 提问（Prompt 技巧）

把大语言模型当成你的私人导师。你提问的水平，就是你认知的天花板。

**万能提问公式：**

```
🎯 角色 + 任务 + 背景 + 输出格式
```

**现成模板（直接复制使用）：**

> **模板① — 学新知识**
> ```
> 你是我的[编程/英语/设计]导师。我是完全零基础的初学者。
> 请用最简单的比喻解释什么是[概念名称]，
> 像教5岁小孩一样讲，不要用专业术语。
> ```

> **模板② — 制定学习计划**
> ```
> 我是一名上班族/学生，每天只有1小时学习时间。
> 请帮我制定一个3个月的[Python/英语/数据分析]学习计划，
> 按周列出目标，每周不超过5小时。
> ```

> **模板③ — 解释复杂内容**
> ```
> 请用最通俗的语言解释[某概念]，
> 并给我3个生活中的类比帮助我理解。
> ```

> **模板④ — 模拟面试**
> ```
> 请扮演面试官，对我进行[Python初级工程师]岗位的模拟面试。
> 先问我10个问题，然后逐题给出答案和解析。
> ```

## 💻 第二步：练一项能"开挂"的硬技能

有台电脑就能学，壁垒高，回报大，未来十年都用得上。

### 🐍 编程 — 学 Python

它能帮你自动化重复工作、爬数据、做分析，甚至写个简单的 AI 应用。

#### Python 学习路线（小白版）

```
📅 第1周：环境搭建 + Hello World
    → 装 Python + VS Code，学会 print()

📅 第2周：变量、数据类型、if 判断
    → 学会存数据、做逻辑判断

📅 第3周：循环、列表、字典
    → 学会批量处理数据

📅 第4周：函数、模块
    → 学会写自己的"工具"

📅 第5周：综合实战
    → 写自动整理文件的脚本
    → 写爬取网页标题的小工具
    → 做简单的数据分析
```

#### 🚗 GitHub Copilot — 你的编程副驾驶

这是你笔记里选中的内容，我重点展开：

**是什么？**
> AI 编程助手，在你写代码时自动帮你补全代码。就像打字时输入法联想下一个词，Copilot 帮你联想下一段代码。

**关键信息：**

| 项目 | 说明 |
|------|------|
| 费用 | 个人版 **$10/月**，学生（有 .edu 邮箱）**免费** |
| 支持编辑器 | VS Code、JetBrains、Neovim 等主流编辑器 |
| 支持语言 | Python、JavaScript、Java、C++ 等主流语言 |
| 核心功能 | 写代码、写注释、写测试、解释代码、修 Bug |
| 适合小白吗 | ✅ **非常适合！它就是你的编程副驾驶** |

**安装步骤（5 分钟搞定）：**

1. 去 https://code.visualstudio.com/ 下载安装 **VS Code**
2. 打开 VS Code，左侧点 Extensions（扩展图标）
3. 搜索 **Python** → 安装微软官方的 Python 插件
4. 再搜索 **GitHub Copilot** → 点击安装
5. 按提示登录 GitHub 账号，开通（有 30 天免费试用）

> [!tip] 小白最简单用法
> 你完全可以用**中文写注释**，Copilot 帮你生成代码：
> ```python
> # 写一个函数，生成随机密码，8位，包含字母和数字
> ```
> 按回车，Copilot 就会自动补全代码，按 `Tab` 接受。

**Copilot vs ChatGPT 怎么选？**

| 工具                 | 适合场景                                     | 怎么配合    |
| ------------------ | ---------------------------------------- | ------- |
| **ChatGPT**        | 问问题、学知识、要完整代码                            | 学新概念时用  |
| **GitHub Copilot** | 写代码时实时补全                                 | 练手写项目时用 |
| **组合使用**           | ChatGPT 教你 → 你在 VS Code 写 → Copilot 帮你提速 | 🚀      |

> 更多细节见 → [[GitHub Copilot 使用指南]]

### 📊 数据分析

> Excel 只是起点。**SQL + Python (Pandas) + Power BI / Tableau** 这一套组合拳，能让你在任何行业都做出数据驱动的决策。这种能力，极其稀缺。

## 🧠 第四步：学英语，把世界变成你的资料库

> 电脑 + 英语 = 解锁全人类的知识宝库，没有语言壁垒。

### 📖 沉浸式阅读

| 工具/方法 | 说明 |
|-----------|------|
| **系统调成英文** | 手机+电脑都切换英文界面，强迫适应 |
| **Readwise** | 同步各平台标注的高亮内容，集中复习 |
| **The Economist** | 经济学人，英语学习的"圣经级"读物 |
| **Medium** | 英文深度博客平台，话题广泛 |

### 🎧 听说重塑

| 工具 | 用途 | 说明 |
|------|------|------|
| **Elsa Speak** | AI 纠音 | 逐句纠正你的发音，像私教 |
| **YouTube 全英文访谈** | 练听力 | 搜你感兴趣的话题，开英文字幕 |
| **Language Reactor** | 浏览器插件 | 看 YouTube / Netflix 显示双语字幕 |
| **italki** | 找语伴 | 和母语者1对1聊天，按次付费 |

**学习步骤：**

```
第1阶段：双语字幕 → 看懂剧情
第2阶段：英文字幕 → 脱离中文
第3阶段：无字幕   → 硬啃
第4阶段：开口说   → italki 找语伴开聊
```

## 🚀 第五步：输出倒逼输入，打造数字资产

> 疯狂提升最好的验证，就是把你学到的，教给别人。

### ✍️ 写下来

开个博客或公众号，学到新概念，就用最简单的话复述一遍。坚持一年，你会发现自己已经成了别人眼里的专家。

**推荐平台：**

| 平台 | 适合 | 特点 |
|------|------|------|
| **微信公众号** | 中文读者 | 私域流量 |
| **知乎** | 知识分享 | 长文友好 |
| **掘金** | 技术文章 | 程序员社区 |
| **Notion / Obsidian 发布** | 个人博客 | 零成本搭建 |

### 🎬 拍出来

做视频或播客，分享你的学习过程、读书心得、项目复盘。这既是在倒逼你深度思考，也是在积累能跟你一辈子的个人品牌。

| 平台 | 适合内容 |
|------|---------|
| **B站** | 教程、Vlog、学习记录 |
| **小红书** | 图文为主，适合分享笔记 |
| **YouTube** | 全球观众，练英语顺便输出 |

### 🔨 做出来

用你学的编程或 AI 工具，写一个解决自己实际问题的小程序。从 0 到 1 发布它，哪怕只有几个用户。这种完整的闭环经验，超过任何纸上谈兵。

**实战点子（从简单开始）：**
1. 写一个自动整理「下载文件夹」的脚本
2. 做一个个人记账页面
3. 用 ChatGPT API 写一个聊天机器人
4. 爬取某个网站的数据做分析
