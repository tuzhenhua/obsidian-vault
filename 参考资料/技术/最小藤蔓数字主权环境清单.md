# 最小藤蔓数字主权环境清单（2026）

> 按"EPROM 精神——用户有最终改写/验证权"的标准，列一份 2026 年还能买到的、三原则全过的硬件。
>
> 不是给所有人的——是给"想自己握着固件"的人的。

---

## 📱 手机 — Google Pixel + GrapheneOS

唯一主流消费设备满足：**BL 可解锁、源码全放、刷完可重锁 Verified Boot**。

| 推荐型号（2026） | 说明 |
|-----------------|------|
| Pixel 9 / 9 Pro / 9 Pro XL / 9a | 首选，7年安全更新，Tensor G4 |
| Pixel 8 / 8 Pro / 8a | 性价比之选，仍完全支持 |
| Pixel 7 / 7 Pro / 7a | 可用，剩余支持年限较短 |
| Motorola Edge 系列（部分） | 与 GrapheneOS 合作机型支持解锁+重锁，方向对但机型少 |
| Nothing Phone (2/2a)、OnePlus 老款、部分 POCO | 官方可解锁 BL、放内核源码，但越来越收紧（等待期、账号绑定），"半开放" |
| Orange Pi Phone / PinePhone / Librem 5 | 全开源 Linux 手机，从 bootROM 起你控制——真正的 EPROM 精神继承者，日常体验粗糙 |

> ⚠️ iPhone、三星国行、华为——基本锁死，不过三原则。

要点：
- 买**非运营商锁定**的国际版
- 刷 GrapheneOS → 重新锁定 Bootloader（防物理篡改）
- 无 Google 服务或沙盒化 Play

### 框架自检

| 原则 | 状态 |
|------|------|
| 可理解 | ✅ 源码全放 |
| 可修理 | ✅ BL 解锁 + 重锁，固件你控 |
| 可自制 | ✅ 沙盒化 Play，权限你定 |

## 📡 网关 — 开源路由器（OpenWrt 主线支持）

### ① OpenWrt One（Banana Pi 合作） — 最尊重用户

- **专为软件自由设计**——永不锁、可救砖、源码全开放
- 官方鼓励你换固件，目前"尊重用户主权"的标杆产品
- SPI NOR/NAND 分开——你可以烧自己固件

### ② GL.iNet Flint 2 (GL-MT6000) — 最平衡

- MediaTek Filogic 830，双 2.5G 口
- 出厂即 OpenWrt 衍生固件，可直接刷原生 OpenWrt
- WireGuard ~900Mbps，适合家庭

### ③ Banana Pi BPI-R4 / R3 — 最开放

- 开源硬件 + 主线 U-Boot + 主线 OpenWrt
- R4：双 10G SFP+、WiFi 7 可选，适合进阶/实验室
- R3：2.5G 双口，便宜紧凑

### ④ Dynalink DL-WRX36（二手）— 最便宜

- Qualcomm IPQ8072A，WiFi 6，OpenWrt 支持极好，约 $50

> ⚠️ 避坑：**Broadcom 芯片**（多数华硕/网件消费款）OpenWrt 无线驱动残缺——**可修理不过。**

### 框架自检

| 原则 | 状态 |
|------|------|
| 可理解 | ✅ 开源固件，你能看 |
| 可修理 | ✅ 自己刷，自己配 |
| 可自制 | ✅ 不绑厂商云，规则你定 |

## 💾 自有服务器 — 数据不出门

### 轻量（入门）：Raspberry Pi 5 (8GB) + NVMe HAT + SSD

- 跑 Pi-hole/AdGuard Home、Vaultwarden、Nextcloud
- 功耗 3-5W，24h 开着

### 全功能（本地 AI 主权首选）：AMD Strix Halo（Ryzen AI MAX+ 系列）

- **Zen 5 + RDNA 3.5 + XDNA 2 NPU 三合一 APU**，统一内存架构
- 最高 **128GB 统一内存**（96GB GPU 可用）——单机能跑 70B 参数 LLM
- 40 CU Radeon 8060S iGPU，**无需独立显卡**也可本地跑大模型
- 50 TOPS NPU，适合持续运行的轻量 AI 任务
- 出现在 HP Z2 Mini G1a、ASUS ROG Flow Z13 等设备中，**可选 Mini PC 形态**
- 对比：运行 Llama 3 70B Q4 时，token/s 是同样 128GB 的 N100 的 **7-10 倍**
- 缺点：功耗较高（45-120W），价格较高（CPU 约 $400-500，+128GB 约 +$400）

> ⚡ **Strix Halo 的价值**：当云 AI 开始要求身份验证时（Anthropic KYC 等），本地能跑 70B 模型的机器就是你的主权底线。

### 性价比之选：Intel N100 Mini-PC (Beelink EQ12/SER5 等)

- 16GB RAM + 512GB NVMe，**约 $150-200**
- Docker 跑 Home Assistant、本地 LLM（Phi-3/Llama 3 8B）、Jellyfin
- x86 兼容性好，可跑 OPNsense 做旁路由
- 适合轻量 AI + 家庭服务器组合

### 框架自检

| 原则 | 状态 |
|------|------|
| 可理解 | ✅ 全栈开源，日志你看得见 |
| 可修理 | ✅ x86 标准化硬件，零件随便换 |
| 可自制 | ✅ Docker 随意组合，不依赖任何云 |

## 框架收束

| 层 | 设备 | 应对什么 |
|----|------|---------|
| 手机 | Pixel + GrapheneOS | 使用权不被拼桌 |
| 网关 | OpenWrt + Pi-hole | 流量不被劫持（摩托罗拉那条线） |
| 服务器 | Mini-PC 自托管 | 数据不出门（AI PC 那条线） |
| 身份 | YubiKey | 追责权——密钥在你手里 |

> **这不是反技术。这是三原则落地到购物清单。** 每一件东西都过得去——可理解、可修理、可自制。过不去的，不买。
