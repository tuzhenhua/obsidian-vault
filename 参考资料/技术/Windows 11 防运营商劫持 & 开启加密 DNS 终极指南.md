
---
**适用系统**：Windows 11 (21H2 及以上)

**目标效果**：防网页弹广告、防DNS劫持、开启 DoH 加密 DNS

**操作耗时**：3 分钟

---

## 一、原理简介（懂了就不怕错）

1. **运营商劫持**：宽带商（电信/移动/联通）会偷偷改你的 DNS 结果，往网页里塞广告或跳转到他们的导航页。
    
2. **DoH (DNS over HTTPS)**：把 DNS 请求伪装成普通的 HTTPS 网页流量（像访问百度一样），运营商看不懂也改不了。
    
3. **IPv6**：新一代互联网协议，速度快、路径短。（详见第五节验证/开启）

---

## 二、准备工作（管理员权限）

1. 右键点击 **开始菜单** → 选择 **终端（管理员）** 或 **Windows PowerShell（管理员）**。
    
2. 如果弹出蓝色或黑色的框，说明成功进入"上帝模式"。
    

---

## 三、执行命令（复制粘贴即可）

请按顺序执行以下命令，**每输完一行按回车**。

### 第 1 步：开启系统全局加密开关

```
netsh dns set global doh=enabled
netsh dns set global dot=enabled
```

### 第 2 步：添加腾讯 DNS (主)

```
netsh dns add encryption 183.60.83.19 "https://doh.pub/dns-query"
```

### 第 3 步：添加阿里 DNS (备)

```
netsh dns add encryption 223.5.5.5 "https://dns.alidns.com/dns-query"
```

### （可选）添加国际 DNS

> 如果你对隐私有更高要求，可以追加以下命令：

```
netsh dns add encryption 1.1.1.1 "https://cloudflare-dns.com/dns-query"
netsh dns add encryption 9.9.9.9 "https://dns.quad9.net/dns-query"
```

| DNS 服务商 | IP 地址 | 特点 |
|-----------|---------|------|
| **Cloudflare** | `1.1.1.1` | 全球最快的 DNS 之一，不记录用户数据 |
| **Quad9** | `9.9.9.9` | 自动拦截恶意/钓鱼网站，安全优先 |

⚠️ 国际 DNS 解析延迟比国内高 30~80ms，日常建议以腾讯/阿里为主，国际 DNS 作为备用。

### 第 4 步：验证命令是否成功

```
netsh dns show global
netsh dns show encryption
```

**✅ 成功标志**：

- `DoH 设置 : enabled`
    
- 列表中能看到 `183.60.83.19`和 `223.5.5.5`以及对应的 `https://`地址。
    

---

## 四、应用 DNS 到网络适配器（Wi-Fi / 以太网 都要做）

⚠️ **注意**：上面的命令只是"装好了武器"，这一步是"拿起武器"。

1. 打开 **设置** → **网络和 Internet** → **Wi-Fi**。
    
2. 点击你当前连接的 Wi-Fi 名称 → **查看其他属性**。
    
3. 找到 **DNS 服务器分配** → 点击 **编辑**。
    
4. 选择 **手动** → 开启 **IPv4**。
    
5. 填入：
    
    - **首选 DNS**：`183.60.83.19`
        
    - **备用 DNS**：`223.5.5.5`
        
    
6. 点击 **保存**。
    

> 💡 如果用的是**有线网（以太网）**，回到"网络和 Internet"页面，选择"以太网"，操作步骤完全一样。**两个都要设置！**

> 💡 如果在第三步添加了国际 DNS，也可以把备用 DNS 换成 `1.1.1.1` 或 `9.9.9.9`。

---

## 五、开启 IPv6（可选）

Windows 11 默认已启用 IPv6，但如果 [test-ipv6.com](https://test-ipv6.com/) 测试不通过，按以下步骤排查：

### 确认 IPv6 已启用

1. 打开 **设置** → **网络和 Internet** → **高级网络设置** → **更多网络适配器选项**。
    
2. 右键你的网卡 → **属性**。
    
3. 确保 ✅ **Internet 协议版本 6 (TCP/IPv6)** 已勾选。
    

### 配置 IPv6 DNS（可选）

在 **设置 → 网络和 Internet → Wi-Fi / 以太网 → DNS 服务器分配** 中：

1. 开启 **IPv6** 开关。
2. 填入：
   - **首选 DNS**：`2402:4e00::`（腾讯 DNSPod）
   - **备用 DNS**：`2400:3200::1`（阿里 DNS）

> ⚠️ IPv6 需要**宽带已开通**且**路由器已开启 IPv6**，否则无法生效。可致电运营商客服免费申请开通。

## 六、效果验证（验收标准）

### 1. 看 IPv6 是否满分

打开浏览器访问：**[test-ipv6.com](https://test-ipv6.com/)**

- ✅ **结果**：显示 **10/10** 且全是绿色。
    
- ✅ **ISP**：显示你的运营商（如 CHINANET / China Telecom）。
    

### 2. 看 DNS 是否加密

打开浏览器访问：**[https://1.1.1.1/help](https://1.1.1.1/help)**

- ✅ 找到 **「Using DNS over HTTPS (DoH)」** 一行，显示 **Yes** 即为加密成功。
- 该页面还会显示你当前使用的 DNS 服务商和是否启用了 ECH（加密 SNI）。

## 七、常见问题（FAQ）

### Q1：需要重启电脑吗？

**答：不需要。** 断开重连 Wi-Fi 即可生效。

### Q2：为什么我在"设置"里手动填 DoH 模板总是报错"无效项"？

**答**：这是 Windows 11 的图形界面 Bug。请用本文的 **PowerShell 命令** 方式配置，100% 成功。

### Q3：会影响网速吗？

**答**：几乎无感。DoH 只是加密了"问路（DNS）"的过程，不影响"走路（下载/游戏）"的速度。反而因为避开了运营商的劣质 DNS，网页打开会更快。

### Q4：国内 DNS 和国际 DNS 怎么选？

**答**：
- **日常使用（推荐）**：腾讯 DNS（`183.60.83.19`）+ 阿里 DNS（`223.5.5.5`），速度最快。
- **隐私优先**：Cloudflare（`1.1.1.1`）+ Quad9（`9.9.9.9`），不记录用户数据。
- **折中方案**：主用国内 DNS，备用设为国际 DNS。
