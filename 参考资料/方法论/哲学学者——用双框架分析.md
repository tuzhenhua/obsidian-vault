# 哲学学者"教士化"的结构性尴尬——用双框架分析

> 原稿：[[参考资料/方法论/哲学学者]]

---

## 认知框架诊断

### 一、本质主义——"这是什么"是结构性陷阱，不是方法选择

文章的元诊断：哲学学者被困在"这是什么"的提问方式里，甩不掉，甚至**在研究反对"这是什么"的哲学家时仍然在用"这是什么"的方式去研究。**

| 文章诊断的层 | 运作方式 |
|:-----------|:--------|
| **提问层** | 任何问题进入学术系统，自动被翻译成"这是什么" |
| **表达层** | 学术会议准入规则过滤"控制不住自己"这类感性表达 |
| **训练层** | 学者的整个训练过程就是"教士化"——你已经不会用别的方式思考了 |

**框架的位置：** 这比[[本质 vs 关系——死的用本质，活的用关系]] 说的盲区更深。框架的盲区是"有时候把活的东西钉死了"——文章说的是**整个系统的提问方式已经被锁死在本质主义上了，不是某个选择的问题。**

### 二、语迷的制度版本——笼子在语言之外

框架原有的语迷说"笼子做成了语言本身"——你连'还有别的说法'这个概念都没有。

这篇文章把语迷推到了**制度层**。你实际上有"别的说法"，甚至知道怎么说（"各位师友，我控制不住自己"），但制度不让你说——说了就破坏准入规则，会被扫地出门。

| 语迷的层 | 笼子在哪 | 你怎么知道你在笼子里 |
|:--------|:--------|:----------------|
| **语言层（框架已有）** | 词汇/语法 | 你说不出笼子外面的词 |
| **制度层（这篇文章）** | 表达规则/准入规则 | 你说得出来但不被允许说 |

**文章说"学者会不假思索地把语言=思想"**——这句话本身就是在拆一个更深层的语迷。**语言=思想**一旦成立，那些没有被语言捕获的感性杂多就自动被排除出"思想"的范畴了。而这种排除不是因为感性杂多不重要——是因为它不能被"这是什么"的格式表达。

### 三、感性杂多 = 认知场的信号

文章用了三个人来举"感性杂多"的例子：

| 来源 | 感性杂多的表现 | 传统哲学的处理 |
|:----|:------------|:------------|
| 奥古斯丁 | "什么是时间？你不问我我知道，你问我我不知道" | 被后来的学者收编为"理论"，丧失了原初的困惑感 |
| 王国维 | "可爱的不可信，可信的不可爱" | 被招安为"拯救与逍遥"的普遍叙事——"这一下子就瞎了" |
| 克拉丽丝 | "好想把他抱回家" | 康德根本不会往这个方向想——这不属于哲学问题 |

框架的 [[本质 vs 关系——死的用本质，活的用关系]] 说关系主义需要"身体在场"，你说的"感性杂多"就是身体在场的操作定义——那些没被理论态度过滤过、直接从身体里长出来的感受。框架有概念（认知场、身体在场）但没有一个专门的概念来描述"这种还没来得及被理论化的原初感受"。**"感性杂多"可以作为框架的一个补丁概念收进来。**

### 四、"超出智力" = 认知场 vs 统计平滑场

文章说：

> "学者如何以甩不掉'这是什么'也得'甩掉'的方式思考？——我认为要超出智力。不要和人工智能比赛智力、推论、概括、文献挑选、归纳总结。你唯一能站住的地方，是超出智力的范围，超出理性囚禁的范围，甚至超出语言的范围。"

这是 [[场——关系主义的操作概念]] 里"认知场 vs 统计平滑场"区分的一个实战版本。统计平滑场擅长的是智力的可量化部分——推论、概括、文献挑选。认知场拥有的是智力之外的东西——感性杂多、身体里的感受、临时生成的判断。文章没说"认知场"这个词，但划出了完全相同的边界。

## 打水漂框架应对

### 五、学术制度场——清晰度极高的藤蔓场

学术哲学场是一个**高清晰度场**：
- **方向**——"这是什么"的理论态度指向哪里，场强就集中在哪
- **场源**——学术会议、期刊审稿、职称评审一起锁定力线方向
- **入场规则**：学术会议不允许你说"各位师友我控制不住自己"
- **惩罚机制**：犯规者会被归为"不专业""不严谨"

三条件满足情况：

| 藤蔓条件 | 在学术哲学场中的运作 |
|:--------|:-----------------|
| 方向替代 | 你接受的是"这是什么"的提问方式——你忘了还可以用不提问的方式思考 |
| 速度加成 | "这是什么"让你写论文更快——分类、定义、论证比露感性杂多更高效 |
| 主体性侵蚀 | 你分不清"学术就该这样"和"学术被训练成只能这样"的区别 |

### 六、这场里的材料类型

| 材料 | 在学术哲学场中的表现 |
|:----|:------------------|
| **铁粉** | 顺着场走，写标准的学术论文——即使你研究的是反对本质主义的海德格尔，你还是用本质主义的论文格式写 |
| **铜** | 不参与学术游戏——不做哲学了 |
| **钕磁铁** | 试图用"感性杂多"的方式做哲学，但不被场识别——像这篇文章自己，74人阅读，在公众号上而不是在学术期刊上 |

**这篇文章的作者自己就是钕磁铁。** 他写这篇文章的方式（感性杂多、东写西写、东拉西扯但没有写散）本身就是在实践"不要问这是什么"的写作方式。但他把文章发在公众号（74人阅读）上，而不是学术期刊上——因为他知道学术期刊的准入规则不接收这种写法。**他用行为证明了自己的诊断：学术界接收不了他写的文章。**

### 七、应对策略——翻、喂、蹲

| 行为 | 在学术哲学场里 |
|:----|:------------|
| **喂** | 继续写标准学术论文——场奖励你，但你的主体性继续被侵蚀 |
| **蹲** | 像文章说的"不知道怎么办"——知道自己被训练成这样了，暂时没有出路 |
| **翻** | 像这篇文章作者做的那样：在学术体制外写公众号、用感性杂多的方式讲道理 |

文章结尾没有给解决方案——"我不知道怎么办"，就是**蹲着**。他知道问题在哪，知道螺帽滑丝了，但他自己拧不动。这个诚实比假装有答案更有价值。

### 八、五指山参照

| 指 | 在学术哲学场里的参照 |
|:---|:-----------------|
| 庄子·虚室生白 | 空掉"这是什么"的提问习惯——从理论态度中暂时退出 |
| 屠名恩·独立 | 选择和学术场保持距离——在体制外写这篇文章本身就是独立的操作 |
| 母亲·安静 | 不教、不追、不评——不用学术标准来要求自己的思考 |
| 屠振华·翻 | 在公众号、随笔、杂文里继续翻——不在学术期刊上翻，在体制外翻 |
| 下一代·待画 | 学术制度的教士化能不可逆到什么程度？下一代学者有没有别的路？ |
