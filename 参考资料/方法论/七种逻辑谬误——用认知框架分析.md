# 七种逻辑谬误——用认知框架分析

> 七种谬误，五种直接命中框架核心。最后一句话提到庄子——"朝三暮四——实质没变，喜怒被说法操控了。"

| 谬误 | 框架对应 |
|:--|:--|
| **后果论证**——"你不想接受这个结果，所以前提是假的" | **身份不允许。** 情绪拦截了判断——不是逻辑不对，是不敢让逻辑跑完 |
| **滑坡论证**——"一旦开了口子就是万丈深渊" | **藤蔓的速度加成。** 跳过中间所有步骤直奔极端。每一步都可以停——但对方不让你停 |
| **诉诸大众**——"这么多人信一定是真的" | **扩音器效应。** 不是在论证——在数人头。人头多的那边不需要论证 |
| **人身攻击**——"你又不是博士凭什么说" | **身份默认。** 不讨论论点，讨论你有没有资格说。标签替你完成了验证审批 |
| **稻草人**——扭曲成容易打倒的版本 | **语迷。** 换了一个词——你的立场就变成了你没说过的东西 |

> 这些谬误有一个共同特征：**它们不是用逻辑说服你——是用情感绕过你的逻辑。** 当情感被调动起来的时候，你的校验系统就关机了。不是在"想"——是在"反应"。
