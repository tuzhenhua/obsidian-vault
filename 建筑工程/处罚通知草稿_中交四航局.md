# 处罚通知（三）：擅自停工（4月27日~28日）

**致：** 中交第四航务工程局有限公司项目经理部

贵司在合同履行过程中，于**2025年4月27日至28日**期间，在未取得发包人同意的情况下擅自停工。根据当日监理日志记录：

> **2025年4月27日：** "工地停工暂无人施工"

**合同依据：** 依据合同专用条款**第16.2.1条**：承包人擅自停工造成不良影响，处合同价款0.5%~5%罚款。

**处罚金额：** **人民币 叁仟元整（¥3,000.00）**

# 处罚通知（七）：项目经理曾湘春未按合同到岗履职

**致：** 中交第四航务工程局有限公司项目经理部

贵司合同约定的项目经理**曾湘春**在施工期间未按合同要求到岗履职，具体违约事实如下：

1. 监理日志明确记载**2025年7月30日**"**管理人员也不在现场**"；
2. 施工期间多个关键节点（基础验收、隐蔽工程验收、工序交接等）项目经理均未到场；
3. 施工期间**多次出现停工、无人施工状态**（已另发处罚通知），作为项目经理未能履行现场管理职责；
4. 经甲方记录，**2025年5月7日**项目经理未在岗（甲方现场记录）。

以上行为违反了专用合同条款关于项目经理在岗的要求：
> 项目经理每周在岗不少于**5天**，每日不少于**8小时**，必须参加项目例会；发包人按月考勤。

**合同依据：**
- 依据合同专用条款**第3.2.2条**：项目经理未经批准擅自离开施工现场，**每次罚款2,000元**；
- 依据合同专用条款**第3.2.3条**：擅自更换项目经理，**每人次罚款10万元**；
- 依据合同专用条款**第3.2.4条**：无正当理由拒绝更换项目经理，**每次罚款3万元**。

**处罚金额：** **人民币 捌仟元整（¥8,000.00）**

鉴于项目经理长期不到岗，贵司应于收到本通知后**7日内**书面说明情况并提交整改方案。若继续不履行在岗职责，我方将依据合同条款处以更高级别处罚并追究由此造成的全部工期及经济损失。

**发包人：** 中国人民解放军国防科技大学军政基础教育学院（盖章）

# 处罚通知（八）：现场安全管理不到位

**致：** 中交第四航务工程局有限公司项目经理部

施工期间，贵司现场安全管理不到位，存在多项安全违规行为：

1. 监理例会上明确指出**"发现工人不带安全帽、安全带的情况较多"**（2025年6月24日监理会议记录）；
2. 施工日志记载现场作业人员安全意识淡薄，安全防护用品佩戴不规范；
3. 作为安全责任主体，贵司未切实履行现场安全管理职责，存在多处安全隐患。

**合同依据：**
- 依据合同专用条款**第6.1.5条**（文明施工）：工人未佩戴安全帽/安全带，**每人次罚款200元**，主管人员处500元罚款。

鉴于违规行为涉及人员较多且监理例会已明确指出仍未彻底整改，按多人次累计及管理不到位综合核定。

**处罚金额：** **人民币 伍仟元整（¥5,000.00）**

请贵司立即加强现场安全教育培训，杜绝安全违规行为再次发生。该款项将从工程款中直接扣除。

**发包人：** 中国人民解放军国防科技大学军政基础教育学院（盖章）
**日期：** 2026年 月 日
