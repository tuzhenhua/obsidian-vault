# 张家界航院宿舍楼——ALC墙板技术要求

**依据图集**：《蒸压加气混凝土砌块、板材构造》13J104

## 基本参数
- **厚度**：100~200mm
- **抗压等级**：A3.5
- **干密度**：≤625kg/m³

## 内墙面做法
ALC墙面板缝构造详参13J104 A6.2相关做法。

## 门窗洞口
附墙暗柱做法和墙面防裂措施同5.3.3节-6。

## 构造柱和圈梁设置
| 条件 | 要求 |
|------|------|
| 平面长度>6m 或 >2倍层高 | 中部加构造柱，间距≤5.0m |
| 框架结构+ALC楼板 | 构造柱间距≤层高且≤4m |
| 洞口宽度>2100mm | 洞口两侧设构造柱 |
| 不同板材/厚度在同一单层 | 不得混用，竖向交接处设构造柱 |
| 墙高大于板挂安装高度 | 门窗洞口或单层高层位置设搁置 |
| 教学楼黑板预埋件等 | 按需设置构造柱 |

## 隔墙与管线一体化
- 管线综合设计基础上实现墙体与管线集成一体化
- 工厂按图纸要求将预埋件提前预埋在ALC墙板中
- **现场开槽埋设管线的不认定为墙体、管线一体化**
