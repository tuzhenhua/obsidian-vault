# 你以为耳机只是听东西，但它还在做另一件事

  

前几天看到一篇文章，一位博主从德国回美国探亲，走进一家沙拉吧，看到一个老人在跟陌生人聊天。他说自己愣了一下——不是觉得这画面多温馨，而是意识到，自己在德国已经很久没见过这种场景了。

然后他往街上一看，明白了：半数人耳朵上挂着 AirPods。你想跟陌生人搭句话，人家根本听不到你。

说实话，人说话少了这件事，我之前就有感觉。但看到一条数据还是没想到：2005年到2019年，美国人日均说话量下降了28%。

28%

这不是"大家变内向了"能解释的幅度

但我的第一反应不是"天哪人怎么不说话了"，而是另一个问题——那些没说出口的话，去哪了？

答案好像很明显：被耳机吃掉了。但仔细想，这件事比"戴耳机所以不说话"深得多。耳机不只是让你听不到别人，它还改变了你听到的东西怎么进入你的脑子。

01

## 你以为耳机只是听东西的工具

先说一个我之前完全不知道的事。

2022年，三位研究者做了一个实验。他们让同一批人听同一段播客，但分成两组——一组戴耳机听，一组用扬声器听。内容完全一样，声音完全一样，唯一的区别是声音从哪里来。

结果：耳机组觉得播客主持人更温暖、更有说服力、更有同理心。扬声器组没有这个感觉。

同样的内容，只是换了播放方式，你对说话者的判断就不一样了。

然后研究者又做了一组：把耳机调成"环绕声"模式，让声音听起来像是从四面八方传来的。结果呢？差异消失了。耳机不再比扬声器更有说服力。

这说明什么？关键不是耳机本身，而是**声音有没有方向**。

|   |   |   |
|---|---|---|
|扬声器<br><br>声音从对面传过来——你知道对面有个人在说话，你在听他讲。"他在说"||耳机<br><br>声音没有方向，没有远近，它就在你耳朵里。你不会觉得"他在说什么"，你会觉得"我听到了什么"。"我在听"|

从"他在说"变成了"我在听"，主语变了。

这就是研究者说的"声音内化"。别人的话，听起来像是你自己的想法。

这不是你被说服了。是你根本没机会"不同意"。

想想你自己戴耳机听播客的时候。一个观点进来，你觉得"嗯有道理"——但你有没有想过，这个"有道理"有多少是因为内容本身，又有多少是因为播放方式让你根本没产生过"不同意"的念头？你分不清。

你以为它只是改变了"你听什么"，但它改变了**"你怎么判断你听到的"**。

  

02

## 耳机不只是让你听不到别人

但还有另一半。

声音内化让你更相信耳机里的声音。但耳机还做了另一件事——它是一块挂在耳朵上的"请勿打扰"牌子。不是你不信任外面的声音，是你根本不给外面机会。

收银员想跟你搭句话，你耳朵里在放播客；邻居想跟你点个头，你耳朵里在放课程。不是你不想理他们，是你根本不知道他们在看你。

美国人日均说话量变化 · 2005–2019

↓ 28%

每年少338个词，一年一年地滑下去。这个数字来自亚利桑那大学的 Matthias Mehl 团队，今年刚发表。Mehl 本人认为，2019年之后，这个数字只会更大。

想想也合理——2020年疫情一来，远程办公成了常态，自结账取代了收银员，导航取代了问路，外卖取代了跟店员的那句"今天有什么推荐"。面对面说话的机会，只会越来越少。

而 Mehl 说了一个细节很重要：这些消失的词，不是少了一段长对话，而是散落在日常里——少了跟收银员的那句话，少了问路的那个瞬间，少了跟邻居的那次点头。

他说，这些小互动"给了社会生活质感"。

你想想自己上次跟陌生人聊天是什么时候？

萨塞克斯大学的 Gillian Sandstrom 研究了一辈子"跟陌生人闲聊"这件事。她的结论是：跟陌生人的闲聊让人更相信"人是好的"。这种信心很难量化，但"人人都需要"。

而且这件事正在加速。44%的美国人已经在用蓝牙耳机。有人每天佩戴超过16个小时——醒着的时间几乎都在戴着。更值得注意的趋势是：开放式耳机——就是那种不塞进耳朵、挂在耳朵外面的——2024年在中国出货量同比翻了一倍多。它的进化方向不是让你听得更清楚，而是让你**永远不用摘下来**。

趋势

耳机正在从"偶尔戴"变成"默认状态"。以前是你想听东西的时候才戴上，现在是你不想听任何东西的时候也戴着——因为摘下来反而觉得不对劲。

当"请勿打扰"变成了默认状态，你不只是挡掉了噪音，你也挡掉了那些让你相信"世界还有善意"的小瞬间。

  

03

## 你连自己的声音也听不见了

但还有一层。

前面说的都是耳机怎么改变你跟别人的关系——更相信耳机里的声音，更不信任外面的声音。但耳机还做了第三件事：它吃掉了你跟自己相处的时间。

你有没有这种感觉——等电梯的时候、走路的路上、排队的间隙，以前这些时候你什么都不做，就那么待着。现在呢？手一伸，耳机戴上，播客打开。这些碎片时间全被填满了。

这看起来好像没什么问题。碎片时间利用起来，多学点东西，不好吗？

问题是，你的大脑不是这么工作的。

默认模式网络（DMN）

简单说，就是发呆的时候，大脑其实在干正事。整理记忆、构建自我叙事、产生创意——深层的反思状态，只有在你不专注于当下任务时才会发生。

然后你想想耳机在干什么。它把每一个"什么都不做"的缝隙都填上了。等电梯的30秒，走路的10分钟，排队的5分钟——以前这些时候你在发呆，现在你在听播客。发呆从默认状态，变成了需要刻意争取的奢侈品。

而且这件事也在加速。AirPods Pro 3已经加了心率传感器和临床级助听功能。佩戴舒适度已经超过音质，成了很多人买耳机的首要考量。它想做的不是让你听东西，是让你永远戴着。

整件事串起来

耳机让别人的声音变成了你脑子里的声音，让你更相信耳机里的话；耳机挡掉了你跟陌生人闲聊的机会，让你更不信任外面的世界；耳机吃掉了你发呆的时间，让你连自己的声音也听不见了。

你不再接触真实的人，也不再接触真实的自己。

那怎么办？扔掉耳机吗？

说实话，我自己也做不到。我每天戴耳机的时间不少于4个小时——听播客、听课程、开会。你让我摘掉，工作都没法干。

但我觉得有一件事是可以做的：**知道它的代价。**

你不知道自己对播客的信任，有一部分是因为声音从你脑子里冒出来的——知道了，下次听到一个"有道理"的观点，你可以多停一秒，想想这个道理是内容本身的，还是播放方式送给你的。

其他的也一样——少了闲聊、少了发呆，知道了，你才有的选。

不是让你扔掉耳机。是让你知道，耳机给你的不只是声音，还有一些你没注意过的东西——你的判断力、你对人的信任、你跟自己相处的机会。

知道了，你才有的选。
