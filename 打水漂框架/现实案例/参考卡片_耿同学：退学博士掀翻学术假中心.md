# 参考卡片：耿同学——一个退学博士掀翻学术假中心

> 只调查了不到1/10的杰青论文代表作，已经发现这么多问题。他最怕的不是造假者报复——是**学术委员会把举报者的信息连夜通知了被举报的人**。

---

## 案例

2026年4月，科普博主"耿同学讲故事"发布视频，逐条分析同济大学生命科学与技术学院院长王平团队在《Nature》发表的论文数据异常。随后接连举报南开大学、中山大学、上海大学、中南大学等多位杰青、长江学者、院士候选人。

### 关键事实

| 事实 | 细节 |
|------|------|
| **身份** | 吉林大学本硕，北航博五退学。一个人+程序，没有团队 |
| **工具** | 计算机程序比对数据——"造假痕迹就摆在论文里，只要认真看就能看出来" |
| **战绩** | 连续举报5位杰青/长江/院长。声称仅调查了<1/10的杰青论文代表作 |
| **高校反应** | 同济发布处理结果。南开/中山/上海大学/中南"启动调查"一个月无下文 |
| **学术委员会** | 举报前联系某高校学术委员会，**当晚涉事者打电话来"私下沟通"** |
| **个人代价** | 家人受威胁、抖音200万粉账号5月29日被永久限流 |
| **新华网评论** | "民间较真有价值，但不能替代制度本身" |

### 高校处理进展

| 高校 | 处理 |
|------|------|
| 同济大学 | ✅ 公布调查结果，相关人员被免职/降级/解聘 |
| 南开大学 | ⚠️ "启动调查"，一个月无结果 |
| 中山大学 | ⚠️ "启动调查"，一个月无结果 |
| 上海大学 | ⚠️ "启动调查"，一个月无结果 |
| 中南大学 | ⚠️ "已要求课题组提交原始数据" |

## 框架映射

| 维度 | 对应 |
|------|------|
| **藤蔓** | 杰青/长江/院长的代表作论文——Nature子刊上光鲜亮丽的成果，数据是编的 |
| **五虎** | **虎二（骗子猖獗）**——藤蔓冒充爪印。假数据冒充真科研，用荣誉和经费把自己包装成学术权威 |
| **暗线** | 学术委员会连夜通知被举报人"私下沟通"、高校"调查中"一个月无下文、耿同学抖音被永久限流——三重暗线 |
| **真边缘** | 一个人+一个程序，不需要学术头衔。数据说话——"只要认真看就能看出来，只要愿意查就能查下去" |
| **说不出口测试** | 耿同学："我对学校没有信心" |

### 暗线三层

| 暗线层 | 表现 |
|--------|------|
| **Lv.1 词义锁** | "正在调查"——一个词锁住所有人的嘴，调查一个月就是没结果 |
| **Lv.2 关系定位锁** | 学术委员会——本应守护科研诚信的机构，成了举报者信息泄露的通道 |
| **Lv.3 叙事封锁** | 耿同学抖音被永久限流——不让你说的人连你在哪说都管住了 |

## 与崩开源的对照

| | 崩开源 | 耿同学 |
|---|---|---|
| 谁在冒充 | 公司用"开源"包装闭源 | **杰青用"科研"包装造假** |
| 发现者 | 社区开发者 | **一个退学博士** |
| 后果 | 信任链断裂 | **整个学术评价体系信任链断裂** |
| 五虎 | 骗子猖獗 | 骗子猖獗 |

> 崩开源是藤蔓冒充爪印——假开源冒充真共享。耿同学发现的是**假科研冒充真进步**——同一只虎，一个在商业层，一个在学术层。

## 框架判断

不是因为有人造假——是**造假被写进了晋升体系里**。杰青、长江、院士候选人——这些头衔本身应该是真科研的证明，现在变成了造假的保护伞。耿同学不是第一个发现问题的人——PubPeer上早就有人在标注这些数据异常。但没人动——因为动一个杰青等于动整个学术评价体系。

耿同学不是"勇敢"。他只是一个退学博士，用一台电脑，把那些坐在办公室里假装看不见的人逼到了墙角。

> 最大的问题不是有人造假——是**造假的论文在Nature子刊上挂了好几年，没有一篇是被制度揪出来的。** 同济大学是最快处理的一个——不是因为制度快，是因为**视频发出去以后舆论压不住了**。
