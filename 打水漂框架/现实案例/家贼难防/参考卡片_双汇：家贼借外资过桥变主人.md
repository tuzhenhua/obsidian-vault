# 参考卡片：双汇——家贼借外资过桥把自己变成主人

> 10年净利510亿，分红517亿。多出来的7亿从以前攒的家底里掏。不是分红——是清算。

---

## 案例

双汇集团原为漯河市100%控股的国企。万隆从国企厂长起步，用不到二十年完成了一场教科书级的管理层收购。

### 操作五步

| 时间 | 动作 | 实质 |
|------|------|------|
| **2003** | 管理层以每股4.7元（市净率仅1.06）拿下双汇发展25%股权 | 低价开口子 |
| **2003-05** | 拿到股权后连续大比例分红，管理层到手2.01亿 | 用国企利润给自己凑收购款 |
| **2006** | 高盛以20.1亿收购双汇集团100%国有股权（当时市值超90亿，PE仅3.3倍） | **贱卖国资**，外资是过桥 |
| **2009** | 高盛原价转给万隆等263人在BVI设立的兴泰集团 | 管理层收购完成 |
| **2016-25** | 10年净利510亿，分红517亿，359亿进了万隆家族海外口袋 | 清算式分红 |

### 七层嵌套架构

```
万隆（最终受益人，份额约49.9%）
  → 兴泰集团（BVI）
    → 雄域投资（BVI）
      → 万洲国际（开曼/港股上市）
        → Glorious Link（BVI）
          → 罗特克斯（香港，持股70.33%）
            → 双汇发展（A股上市）
```

六层海外壳，一层境内上市公司。穿透到底就一个人。

### 附带伤害

- 分红掏空了企业：2025年日均分红660万，全年只留8100万发展
- 品控崩了：承诺15年的"头头检"沦为空谈，林可霉素超标37.5倍
- 内斗：2021年长子万洪建实名举报父亲通过关联交易给双汇造成超8亿损失

## 框架映射

| 维度 | 对应 |
|------|------|
| **五虎** | **虎三（家贼难防）**——不是外部藤蔓，是自己人用制度工具把国企挖空 |
| **垄断四步** | 平等（国企全民所有）→ **占住**（低价拿25%股权）→ **定规矩**（七层架构、境外控股）→ **付费/求人**（清算式分红，利润全抽走） |
| **三权侵蚀** | **所有权**被架空——名义上曾是国企，实质上通过股权运作把所有权层层转移到了BVI；**设计权**全程在管理层——七层嵌套、分红比例、品控投入全是他们定的 |
| **藤蔓绑法** | **示能**（把企业做大，你功不可没）→ **分化**（低价给你股权，"你也是股东了"）→ **取代**（国企变成私企，利润全抽境外） |
| **暗线** | 六层BVI壳——穿透极难。外资高盛当过桥，让"贱卖国资"看起来像"市场化运作"。七层嵌套本身就是暗线——你以为你看到了上市公司，你看不到BVI后面是谁 |

## 与国有经济那篇的对照

| | 国有经济 | 双汇 |
|---|---|---|
| 手法 | 政策倾斜、行业准入 | **管理层收购 + 外资过桥 + 清算式分红** |
| 五虎 | 家贼难防 | 家贼难防 |
| 关键差异 | 制度性垄断 | **个人把国企变成了私企** |
| 暗线 | Lv.1-2 | **Lv.2-3（七层BVI穿透极难）** |

> 国有经济那篇讲的是制度怎么偏袒国企。这篇讲的是**国企管理者怎么用制度工具把国企变成自己的**。同一个虎，两个方向——一个从外往里抢，一个从里往外搬。

---

## 框架判断

不是国有资产流失——是**国有资产的合法化转移**。每一步都有合同、有审批、有评估报告。低价拿股权叫"市场化改革"，外资过桥叫"引进战略投资者"，七层嵌套叫"国际资本运作"，清算式分红叫"回馈股东"。

最狠的不是偷——是**每一步都合法，合起来就是一场完美的抢劫**。

> 如果把双汇留在国资手里，10年分红500多亿归漯河市财政。卖了之后，一分钱都没了，还倒贴了品控和食品安全。
>
> 家贼难防不是因为他们藏得深——是因为他们拿着你的钥匙，穿着你的制服，走的是你修的路。
