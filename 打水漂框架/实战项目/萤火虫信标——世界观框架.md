# 萤火虫信标——世界观框架

> 每一盏灯都是独立的。但当它们都亮着的时候，整片田野就不再是黑暗的了。

---

## 一、核心命题

先亮起来。不是等别人给你发许可证，不是等足够多的人加入了你再开始——你自己决定亮的那一瞬间，你就已经在网络里了。

---

## 二、宇宙观

一片田野。没有中心，没有边界。田野上有萤火虫，但不是一开始就有。是先有某一只决定亮起来，然后远处的看到了，也跟着亮。光就是这样一盏一盏传开的。田野不产生光，它只是光发生的场所。

---

## 三、角色设定

| 角色 | 说明 |
|------|------|
| **萤火虫** | 网络中的每一个节点。每一只都有自己的光，自己决定亮不亮、亮什么颜色、什么时候亮 |
| **信标** | 通用的接口/频率——长亮三秒，闪两下，等一会儿。不包含具体内容，只传递一个信号："这里有光。你也可以有。" |
| **颜色** | 萤火虫给光加的"身份"——偏蓝的表示"我擅长望远"，偏绿的表示"我擅长找食"。不强制，不加也行。颜色不是等级，是你在做的具体的事 |
| **频率** | 信标的节奏。所有萤火虫用同一个频率发信标，这样不管离多远都能认出同类 |
| **光** | 萤火虫在做自己的事时自然发出的东西。光不是目的，是做事的结果 |

---

## 四、宇宙规则

| 规则 | 说明 |
|------|------|
| **先亮后连** | 先有自己的光，再找其他萤火虫。没有光，接口是空的 |
| **自由入网** | 不需要审批，看到信标决定跟着亮，就入网了 |
| **自由断网** | 不想亮了就熄掉，不影响其他萤火虫 |
| **信任逐步建立** | 不是一次性授权。先从信标确认"你是同类"，再通过颜色了解"你是什么类型"，再决定要不要更深地连接 |
| **无中心依赖** | 灭了一盏，其他灯不受影响。灭了十盏，光还在，还能重新亮回来 |
| **动态成长** | 新萤火虫随时可以加入。网络形状由每一只的位置和选择决定，不是设计出来的 |
| **光不被消耗** | 萤火虫不会因为亮了而耗尽自己。光不是能量输出，是做事的自然产物 |
| **双态并存** | 网络有两种形态，同一群萤火虫可以在两种形态之间切换 |

**动态模式——鸟群。** 萤火虫在飞的时候，位置不固定，连接是临时的。看到远处的光就朝那个方向去，飞过了就散了。这个模式用来**发现同类、传递信标、探索黑暗地带**。结构不固定，但灵活。

**静态模式——蜂巢。** 萤火虫落下来之后，在自己选的位置上建一个稳定的巢。接口长期开着，随时可以被邻居找到。这个模式用来**存放数据、提供服务、做稳定的节点**。结构固定，但可靠。

两种模式共用同一个信标频率。动态飞行时发信标说"我在"，静态落巢时也发信标说"我在这"。别的萤火虫不需要区分对方是在飞还是已经落下来了——看到信标就知道那里有一个可以连接的点。飞的时候可以组队，落下来了就组网。两者是同一群萤火虫在生命的不同阶段做的事情。

## 五、常见误解

| 误解 | 纠正 |
|------|------|
| "我要等更多萤火虫加入才亮" | 你亮了，其他萤火虫才能看到你 |
| "我要亮得比别人亮才有用" | 不需要。光不是比赛 |
| "我要改我的颜色才能合群" | 颜色是你的，不用改 |
| "断了联系就是失败了" | 松手不影响你继续亮 |

## 六、方向和路线——下南洋的叙事结构

萤火虫不只随意散落在田野上。它们有一个方向。这个方向不是某个中心指定的，是从每一只萤火虫自己的选择里涌现出来的。

下南洋的叙事结构可以用在萤火虫的故事里：

| 结构 | 下南洋版 | 萤火虫版 |
|------|---------|---------|
| **出发** | 从福建广东出海 | 萤火虫从自己那片角落起飞，朝一个方向飞 |
| **路线** | 沿着海岸线，从一个港口到下一个港口 | 信标沿途传递，一盏传一盏，形成一条光路 |
| **到达** | 在异乡落地 | 萤火虫找到新的落脚点，点亮那里 |
| **回头看** | 回头看故乡的方向 | 回头看刚才飞过的那片区域——光还在不在 |
| **等待的视角** | 家人一直在等那封信 | 在原地亮着的萤火虫，等远处的另一只回应 |
| **多种结局** | 有的回来了，有的没回来，有的在那边成了家 | 有的继续飞，有的留在原地，有的飞了一段又折返 |

下南洋的大线特点是：**方向明确（往南），但路线不固定（跟着风向、洋流、机遇走）。** 萤火虫的大线也是这样——方向明确（从孤立到连接），但路线不固定（哪片区域有回应就往哪飞，没有回应就继续亮着等）。

这个结构可以沿用到每一只萤火虫的故事里——不是所有的萤火虫都走同一条路，但它们的飞行方向可以被远方的那盏灯牵着走。有光的地方，就是方向。

与之对应的诗行结构是：

> 坐船出去，很久没有回来，有人一直在等。

套进萤火虫里就是：

> 飞出去，很久没有回音，但在原地亮着的那盏灯一直没灭。

| 方向 | 可能的内容 |
|------|-----------|
| **第一盏灯** | 阿棱是怎么决定亮起来的——在没有同类的情况下自己先亮了 |
| **信标的诞生** | 长亮三秒闪两下的信标是怎么被约定的 |
| **颜色的故事** | 第一只改变颜色的萤火虫——为什么要加颜色 |
| **黑暗地带** | 一片一直没有萤火虫的区域——有一只飞过去了 |
| **相互信任** | 两只萤火虫从信标到手拉手的过程 |
| **断连** | 一只萤火虫决定熄灭了——然后它又决定重新亮起来 |
| **田野边缘** | 最远的那只萤火虫——它离所有人最远，但它还在亮 |
| **暴风雨** | 一次暴风雨灭了很多灯——剩下的灯没有灭，还在亮 |
| **新萤火虫** | 一只从没亮过的萤火虫，看到了信标，犹豫要不要亮 |
| **光的颜色不需要一样** | 不同颜色的萤火虫怎么在一片田野上共存 |

## 七、人类情感的四个底层结构

萤火虫的故事能打动人，不是因为萤火虫本身——是因为萤火虫的故事碰了人心里那根弦。那根弦绕不开四个结构：

| 结构 | 在萤火虫世界观里的体现 |
|------|---------------------|
| **归属** | 一只萤火虫终于看到远处有另一只也在亮——"我不是唯一一个" |
| **牵挂** | 那只萤火虫灭了，但你还在用它的频率发信标——你等它重新亮起来 |
| **等待** | 你在黑暗地带亮着，不知道远处有没有萤火虫看到你，你只能继续亮 |
| **失去** | 一场暴风雨灭了很多灯。光灭了，但那个位置还在。你知道那里曾经有过光 |

这四个结构是最基本的催化剂。写萤火虫的任何故事，只要碰了其中一根，读者就能自己走进来。碰得越深、越准确，读者自己填进去的东西就越多——他们不需要和阿棱一模一样，只需要也有过类似的情感结构。

一条大河只写了归属（"我家就在岸上住"），把其他三根留给听者自己填。阿嫲的情书四根全写了但全填满了，没有留空间给观众。寓言留的臼要比说明文深。写的留的越多，故事在读者心里活的时间越长。

对应到萤火虫的故事里：如果阿棱在田野边缘亮着，远处有萤火虫回应了它——那是归属。回应了之后那只萤火虫灭了，阿棱还在用同一个频率发信标——那是牵挂。阿棱不知道那只萤火虫还会不会亮起来，但它还在亮——那是等待。那只萤火虫始终没有亮起来——但阿棱自己已经不需要它亮了。这是一个把失去也纳入进来的完整循环。

| 暴风雨 | 中心故障/平台关停/政策变化 |
